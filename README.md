## Jak začít

1. **Přehled roku** → `00 - Přehled a plán.md`
2. **Co učit a kdy** → `Matematika/Sekvence hodin/[ročník].md` nebo `Informatika/Sekvence hodin/`
3. **Příprava na hodinu** → `Matematika/Přípravy na hodinu/00 - Šablona přípravy.md`
4. **Test pro žáky** → `Matematika/Testy/[ročník] - Testy.md`
5. **Učebnice** → `Učebnice a materiály.md` (Prometheus, Fraus, stránkové reference)
6. **Nápady na hry a procvičování** → `Nápady na procvičování.md`

## Klíčové poznámky

| Téma | Poznámka |
|---|---|
| **RVP ZV 2023 – Informatika** | Velká reforma: informatika je nově povinná na celém 2. stupni, zaměřena na algoritmické myšlení a programování (Scratch, Python, HTML). Kancelářský software není těžiště. |
| **JPZ (Jednotná přijímací zkouška)** | Od března 9. ročníku věnovat většinu hodin matematiky přípravě na JPZ. Testy zdarma na [cermat.cz](https://cermat.cz). Bez kalkulačky. |
| **Učebnice – Prometheus** | Modrý rámeček = definice, žlutý = vzorový příklad, bílý = procvičení, červené hvězdičky = rozšíření. Sbírka úloh (SÚ) je nezbytný doplněk. |
| **Hejného matematika** | Někteří žáci 6. ročníku mohli přijít z 1. stupně s Hejného metodou – konstruktivistický přístup bez algoritmů. Je dobré s tím počítat při úvodu. |
| **Sekvence hodin – reference stránek** | Reference „SÚ str. X, č. Y–Z" jsou orientační; ověřte s konkrétním vydáním učebnice, které škola používá. |
