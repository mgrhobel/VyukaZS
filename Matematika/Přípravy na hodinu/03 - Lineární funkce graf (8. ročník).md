## PRŮBĚH HODINY

### 1. ÚVOD – Opakování (5 min)

**Kvíz (ústně):**
- Co je funkce? (Každému x přísluší právě jedno y)
- Co je definiční obor? Co je obor hodnot?
- Funkce y = 3x: pro x = 2, y = ? (6)

**Přechod k dnešnímu tématu:**
> „Umíme spočítat funkční hodnoty. Dnes je budeme zakreslovat do grafu."

---

### 2. VÝKLAD – Postup sestrojení grafu (15 min)

**Krok za krokem na tabuli – y = 2x − 1:**

```
Krok 1: Narýsuj souřadnicový systém (osy x, y; popis; stupnice)

Krok 2: Vytvoř tabulku funkčních hodnot (min. 3 body, ideálně 5)
  x:   −2   −1    0    1    2
  y:   −5   −3   −1    1    3

Krok 3: Zaznač body do souřadnicového systému
  A = (−2; −5), B = (−1; −3), C = (0; −1), D = (1; 1), E = (2; 3)

Krok 4: Spoj body přímkou (ne jen úsečkou – přímka přes celý obrázek)

Krok 5: Popis – napiš k přímce předpis: y = 2x − 1
```

**Výsledný graf:** Přímka stoupá zleva doprava (a = 2 > 0 → stoupající funkce)

```
STOUPAJÍCÍ vs. KLESAJÍCÍ:
  a > 0: stoupající (y roste, jak x roste)
  a < 0: klesající  (y klesá, jak x roste)  
  a = 0: konstantní (y = b, vodorovná přímka)
```

**Druhý příklad (žáci doplňují tabulku, učitel nakreslí):**
```
y = −x + 3

  x:   0    1    2    3    4
  y:   3    2    1    0   −1
  
Průsečík s osou y: x = 0 → y = 3, bod (0; 3)
Průsečík s osou x: y = 0 → 0 = −x + 3 → x = 3, bod (3; 0)
```

---

### 3. PRÁCE NA GRAFU – žáci sami (12 min)

**Zadání (na tabuli, žáci do sešitu):**

Nakresli grafy těchto funkcí do JEDNOHO souřadnicového systému (různé barvy!):
1. y = x + 2  (modrá)
2. y = −2x + 4  (červená)
3. y = 3  (zelená – konstantní)

Pro každou funkci:
- Vyplň tabulku (x od −2 do 4)
- Zaznač body
- Nakresli přímku
- Urči: stoupající / klesající / konstantní

**Klíč:**
1. Stoupající, průsečík s y: (0; 2), s x: (−2; 0)
2. Klesající, průsečík s y: (0; 4), s x: (2; 0)
3. Konstantní – vodorovná přímka y = 3

---

### 4. ČTENÍ Z GRAFU (8 min)

**Na tabuli předkreslit grafy, žáci určí předpis:**

```
Graf A: přímka prochází (0; −2) a (1; 1)
  → směrnice: a = (1−(−2))/(1−0) = 3; b = −2 → y = 3x − 2

Graf B: přímka prochází (0; 4) a (2; 0)
  → a = (0−4)/(2−0) = −2; b = 4 → y = −2x + 4
```

**Klíčový vzorec (na tabuli):**
```
Směrnice: a = (y₂ − y₁) / (x₂ − x₁)
```

---

### 5. ZÁVĚR + DÚ (5 min)

**Shrnutí:**
- Postup: tabulka → body → přímka → popis
- Stoupající ↔ a > 0; klesající ↔ a < 0
- Průsečíky: s y: x=0; s x: y=0

**DÚ:** SÚ str. 78, č. 1–5 (sestrojení grafů)

**Exit ticket:**
> „Funkce y = −3x + 6: Kde protíná osu y? Kde osu x? Je stoupající nebo klesající?"
> (Osa y: (0; 6); osa x: (2; 0); klesající)
