## PRŮBĚH HODINY

### 1. ÚVOD – Opakování a motivace (5 min)

**Rychlé opakování (ústně):**
```
Co je čitatel a jmenovatel?   (čitatel/jmenovatel)
Zkraťte: 8/12 =              (2/3)
Rozšiřte 3/4 na jmenovatel 12: (9/12)
Kolik je NSN(4, 6)?           (12)
```

**Motivace:**
> „Snědl jsem 1/4 pizzy a pak ještě 1/4. Kolik pizzy jsem celkem snědl?"
> „Co kdybych snědl 1/3 pizzy a pak 1/4?"

---

### 2. VÝKLAD – Sčítání zlomků se STEJNÝM jmenovatelem (8 min)

**Pravidlo:** Sčítáme/odčítáme pouze čitatele. Jmenovatel se NEMĚNÍ.

```
a/c + b/c = (a + b)/c

Příklady:
  1/5 + 2/5 = 3/5         (1 pětina + 2 pětiny = 3 pětiny)
  7/8 − 3/8 = 4/8 = 1/2   (vždy krátit výsledek!)
  5/6 + 4/6 = 9/6 = 3/2 = 1 1/2  (výsledek > 1 → smíšené číslo)
```

**Vizualizace na tabuli:** Nakreslit koláč dělený na 8 dílků, vybarvit 5 a pak 4 → celkem 9/8 = 1 1/8

**Žáci počítají (ústně):**
```
3/7 + 2/7 =     (5/7)
9/10 − 4/10 =   (5/10 = 1/2)
5/4 + 3/4 =     (8/4 = 2)
```

---

### 3. VÝKLAD – Sčítání zlomků s RŮZNÝMI jmenovateli (12 min)

**Postup ve 4 krocích:**

```
Krok 1: Najít NSN obou jmenovatelů → společný jmenovatel (SJ)
Krok 2: Rozšířit oba zlomky na SJ
Krok 3: Sečíst/odečíst čitatele (jmenovatel se nemění)
Krok 4: Zkrátit výsledek (pokud je to možné)
```

**Příklad 1:** 1/4 + 1/6
```
Krok 1: NSN(4, 6) = 12
Krok 2: 1/4 = 3/12;   1/6 = 2/12
Krok 3: 3/12 + 2/12 = 5/12
Krok 4: 5/12 nelze zkrátit  →  výsledek: 5/12
```

**Příklad 2:** 5/6 − 1/4
```
Krok 1: NSN(6, 4) = 12
Krok 2: 5/6 = 10/12;   1/4 = 3/12
Krok 3: 10/12 − 3/12 = 7/12
Krok 4: 7/12 nelze zkrátit  →  výsledek: 7/12
```

**Příklad 3 (smíšená čísla):** 2 1/3 + 1 3/4
```
Způsob 1 – přes zlomky:
  2 1/3 = 7/3;   1 3/4 = 7/4
  NSN(3, 4) = 12;  7/3 = 28/12;  7/4 = 21/12
  28/12 + 21/12 = 49/12 = 4 1/12

Způsob 2 – přes celé a zlomkové části:
  Celé: 2 + 1 = 3
  Zlomky: 1/3 + 3/4 = 4/12 + 9/12 = 13/12 = 1 1/12
  Celkem: 3 + 1 1/12 = 4 1/12  ✓
```

**Příklad 4 (odčítání s převodem):** 3 1/6 − 1 3/4
```
NSN(6, 4) = 12
Zlomky: 1/6 = 2/12;   3/4 = 9/12
Problém: 2/12 < 9/12 → musím si půjčit 1 celé od 3:
  3 2/12 = 2 + 1 + 2/12 = 2 + 14/12
  2 14/12 − 1 9/12 = 1 5/12
```

### 4. FRONTÁLNÍ PROCVIČOVÁNÍ (8 min)

Žáci počítají v sešitech, jdou k tabuli:

| Příklad | Výsledek |
|---|---|
| 2/9 + 5/9 | **7/9** |
| 11/12 − 5/12 | **6/12 = 1/2** |
| 1/3 + 1/4 | **7/12** |
| 3/4 − 2/5 | **7/20** |
| 1/2 + 1/3 + 1/6 | **6/6 = 1** |
| 2 3/5 + 1 2/3 | **4 4/15** |

---

### 5. SKUPINOVÁ AKTIVITA – „Zlomková štafeta" (7 min)

Třída se rozdělí na skupiny po 4. Každá skupina dostane list s řetězcem výpočtů:

```
Skupina A:
Start: 1/2
+ 1/3 =        (5/6)
− 1/4 =        (7/12)
+ 5/12 =       (1  → celé číslo!)

Skupina B:
Start: 3/4
− 1/6 =        (7/12)
+ 5/12 =       (1)
− 1/3 =        (2/3)

Skupina C:
Start: 2/5
+ 1/2 =        (9/10)
+ 1/10 =       (1)
− 3/8 =        (5/8)
```

Každý člen skupiny vypočítá jeden krok a předá dál. Soutěž o správnost.

---

### 6. ZÁVĚR + EXIT TICKET (5 min)

**Shrnutí:**
- Stejný jmenovatel → sčítám/odčítám jen čitatele
- Různé jmenovatele → nejdřív NSN → rozšířit → pak počítat
- Výsledek vždy zkrátit!

**Domácí úkol:** SÚ str. 56–57, č. 40–62 (H30 + H31)

**Exit ticket:**
```
1. Vypočítej: 3/8 + 5/8 =               (1 = 8/8)
2. Vypočítej: 2/3 − 1/4 =              (5/12)
3. Vypočítej: 1 1/2 + 2/3 =            (2 1/6)
```
