## PRŮBĚH HODINY

### 1. ÚVOD – Motivace a číselná osa (10 min)

**Motivační scénáře (na tabuli, žáci diskutují):**

> „Kde se setkáváme se záporným číslem v každodenním životě?"

```
– Teplota:        −5 °C (5 stupňů pod nulou)
– Bankovní účet: −200 Kč (dluh 200 Kč)
– Nadmořská výška: −45 m (45 metrů pod mořem)
– Výtah:          podlaží −2 (druhé podzemní podlaží)
– Skóre:          −3 body (pokuta ve hře)
```

**Číselná osa – kresba na tabuli:**

```
←─────────────────────────────────────────────────→
  -7  -6  -5  -4  -3  -2  -1   0  +1  +2  +3  +4
              záporná         │        kladná
                            NULA
```

Klíčové pojmy:
- **Záporné číslo** – vlevo od nuly, píšeme se znaménkem „−"
- **Kladné číslo** – vpravo od nuly (znaménko „+" většinou nepíšeme)
- **Celá čísla** = záporná + nula + přirozená
- **Absolutní hodnota** |a| = vzdálenost od nuly: |−5| = 5, |+3| = 3, |0| = 0

Žáci zaznačí na svou vytištěnou číselnou osu: −4, −1, 0, 2, −7, 5

---

### 2. VÝKLAD – Porovnávání celých čísel (5 min)

> **Pravidlo:** Na číselné ose je větší číslo vždy VPRAVO.

```
−7 < −3 < 0 < 2 < 5     (i když |−7| > |−3|!)

Pozor na typickou chybu:
  „−10 je větší než −2, protože 10 > 2"  ← ŠPATNĚ!
   Na číselné ose je −10 daleko vlevo → −10 < −2  ✓
```

**Žáci seřadí (na tabuli):** −5, +3, 0, −8, +1, −3
> Řešení: −8, −5, −3, 0, +1, +3

---

### 3. VÝKLAD – Sčítání a odčítání celých čísel (10 min)

**Sčítání – pravidla:**

```
Stejná znaménka → sečti, zachovej znaménko:
  (−3) + (−5) = −8     (oba záporné → výsledek záporný)
  (+4) + (+6) = +10

Různá znaménka → odečti menší |hodnotu| od větší, výsledek má znaménko "vítěze":
  (−7) + (+4) = −3     (|−7| > |+4|, vítěz je −7)
  (+9) + (−3) = +6     (|+9| > |−3|, vítěz je +9)
```

**Odčítání – klíčové pravidlo:**

```
a − b  =  a + (−b)      ← odčítání = přičtení čísla opačného

  5 − (−3) = 5 + 3 = 8        (dvě mínuska = plus)
  −4 − (−2) = −4 + 2 = −2
  −4 − (+3) = −4 + (−3) = −7
```

**Příklady na tabuli (žáci počítají):**
```
a) (−3) + (−4) = −7
b) (−8) + 5 = −3
c) 6 − (−2) = 8
d) −5 − 3 = −8
e) (−7) − (−7) = 0
```

### 4. FRONTÁLNÍ PROCVIČOVÁNÍ (10 min)

Žáci zapisují výsledky do sešitu, volají odpovědi:

| Příklad | Výsledek |
|---|---|
| (−4) + (−6) | **−10** |
| (−12) + 7 | **−5** |
| 8 + (−15) | **−7** |
| (−3) − (−8) | **+5** |
| (−6) − 4 | **−10** |
| 0 − (−9) | **+9** |

**Slovní úlohy (2 příklady):**
```
1. Ráno bylo −8 °C. Do odpoledne se teplota zvýšila o 12 °C. Jaká je odpolední teplota?
   → −8 + 12 = 4 °C  ✓

2. Na účtu mám −200 Kč (dluh). Dostanu 350 Kč výplatu. Kolik bude na účtu?
   → −200 + 350 = +150 Kč  ✓
```

### 5. INDIVIDUÁLNÍ PRÁCE – Pracovní list (7 min)

**Část A – Číselná osa:**
```
1. Zakresli na číselnou osu: −6, −2, 0, 4, −9, 3
2. Seřaď od největšího k nejmenšímu: −4, 2, −7, 0, 5, −1
3. Vypočítej absolutní hodnotu: a) |−15|   b) |+8|   c) |0|   d) |−1|
```

**Část B – Výpočty:**
```
a) (−5) + (−9) =          b) (−6) + 11 =
c) 3 + (−14) =            d) (−8) − (−3) =
e) −7 − 5 =               f) 12 − (−4) =
```
*(Výsledky: −14; +5; −11; −5; −12; +16)*

**Část C – Pro rychlé žáky:**
```
Kombinované výrazy:
a) (−3) + (−4) − (−2) + 5 =     (výsledek: 0)
b) −12 + 7 − (−5) − 8 =         (výsledek: −8)

Slovní: Ponorka je 45 m pod mořem (−45 m). Klesne o 23 m.
        Pak stoupne o 38 m. V jaké hloubce je?
        → −45 − 23 + 38 = −30 m  (30 m pod mořem)
```

### 6. ZÁVĚR + EXIT TICKET (3 min)

**Shrnutí pravidel (žáci říkají):**
- Záporná čísla jsou vlevo od nuly na číselné ose
- Větší číslo = více vpravo (−2 > −7)
- Odčítání = přičítání opačného čísla
- |−a| = a (absolutní hodnota je vždy kladná nebo nula)

**Domácí úkol:** SÚ str. 35–38, č. 1–30 (výběr dle výkonu)

**Exit ticket:**
```
Vypočítej: (−6) + (−4) − (−5) =
                                         (výsledek: −5)

Seřaď od nejmenšího: −3, 1, −8, 0, −1
                                         (−8, −3, −1, 0, 1)
```
