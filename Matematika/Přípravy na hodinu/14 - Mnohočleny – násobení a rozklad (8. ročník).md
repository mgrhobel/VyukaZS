## PRŮBĚH HODINY

### 1. OPAKOVÁNÍ – Sčítání a odčítání mnohočlenů (5 min)

**Klíčové pravidlo:** Sčítám POUZE stejnorodé členy (stejná proměnná + stejný mocnitel).

```
(3x² + 2x − 5) + (x² − 4x + 3) = 4x² − 2x − 2

Záporka před závorkou = otočit VŠECHNA znaménka:
  − (2x − 3) = −2x + 3

(5x² − 3x + 1) − (2x² + x − 4) = 3x² − 4x + 5
```

Žáci spočítají 2 příklady ústně.

---

### 2. VÝKLAD – Násobení mononomu × mnohočlen (7 min)

**Pravidlo:** Každý člen závorky vynásobím monomomem.

```
2x(3x² − x + 4)
= 2x × 3x²  +  2x × (−x)  +  2x × 4
= 6x³ − 2x² + 8x

−3x²(2x − 5)
= −6x³ + 15x²
```

**Připomenutí pravidel mocnin:**
```
x^a × x^b = x^(a+b)
x × x² = x³;   2x × 3x² = 6x³
```

**Procvičení:**
```
a) 4x(x² − 3x + 2) =      (4x³ − 12x² + 8x)
b) −2x²(3x − 1) =         (−6x³ + 2x²)
c) 5(2x² − x + 3) =       (10x² − 5x + 15)
```

---

### 3. VÝKLAD – Násobení mnohočlen × mnohočlen (8 min)

**Pravidlo (FOIL nebo „každý s každým"):**

```
(a + b)(c + d) = ac + ad + bc + bd

Příklad 1: (x + 3)(x − 5)
= x·x  + x·(−5)  + 3·x  + 3·(−5)
= x²  − 5x  + 3x  − 15
= x² − 2x − 15

Příklad 2: (2x + 1)(3x − 4)
= 6x²  − 8x  + 3x  − 4
= 6x² − 5x − 4

Příklad 3: (x + 2)(x² − x + 3)
= x³ − x² + 3x + 2x² − 2x + 6
= x³ + x² + x + 6
```

**Procvičení:**
```
a) (x + 4)(x − 4) =        (x² − 16)  ← pozor na tento tvar!
b) (3x − 2)(x + 5) =       (3x² + 13x − 10)
c) (x − 1)(x² + x + 1) =   (x³ − 1)   ← zajímavý vzorec!
```

---

### 4. VÝKLAD – Algebraické vzorce (8 min)

**Tři základní vzorce (memorovat!):**

```
(a + b)²  = a² + 2ab + b²     ← čtverec součtu
(a − b)²  = a² − 2ab + b²     ← čtverec rozdílu
(a + b)(a − b) = a² − b²      ← rozdíl čtverců
```

**Odvození (a + b)² vizuálně na tabuli:**
```
Čtverec se stranou (a + b):
┌────┬────┐
│ a² │ ab │
├────┼────┤
│ ab │ b² │
└────┴────┘
Celkový obsah: a² + ab + ab + b² = a² + 2ab + b²  ✓
```

**Příklady – rozvinutí:**
```
(x + 5)² = x² + 10x + 25
(2x − 3)² = 4x² − 12x + 9
(3x + 2)(3x − 2) = 9x² − 4
(x + 1)(x − 1) = x² − 1
```

**Příklady – rozklad (zpětně):**
```
x² − 16 = (x + 4)(x − 4)
x² + 6x + 9 = (x + 3)²
4x² − 20x + 25 = (2x − 5)²
```

---

### 5. VÝKLAD – Rozklad mnohočlenu na součin (5 min)

**Tři metody rozkladu:**

```
1. Vytýkání (společný faktor):
   6x² + 9x = 3x(2x + 3)        (vytknout 3x)
   4x³ − 8x² + 12x = 4x(x² − 2x + 3)

2. Pomocí vzorců:
   x² − 25 = (x + 5)(x − 5)     (rozdíl čtverců)
   x² + 8x + 16 = (x + 4)²      (čtverec součtu)

3. Kombinace:
   2x² − 8 = 2(x² − 4) = 2(x + 2)(x − 2)
   3x² − 12x + 12 = 3(x² − 4x + 4) = 3(x − 2)²
```

---

### 6. PROCVIČENÍ + EXIT TICKET (12 min)

**Individuální práce – smíšené úlohy:**
```
Rozviňte:
a) (x + 6)² =                   (x² + 12x + 36)
b) (4x − 1)² =                  (16x² − 8x + 1)
c) (x + 3)(x − 3) =             (x² − 9)
d) 3x(x² − 4x + 1) =            (3x³ − 12x² + 3x)

Rozložte na součin:
e) x² − 49 =                    ((x+7)(x−7))
f) x² − 6x + 9 =                ((x−3)²)
g) 5x² + 15x =                  (5x(x+3))
h) 2x² − 18 =                   (2(x+3)(x−3))
```

**Párová kontrola:** Soused zkontroluje výsledky.

**Exit ticket:**
```
a) Rozviňte: (2x − 5)²                    (4x² − 20x + 25)
b) Rozložte na součin: x² − 10x + 25      ((x−5)²)
c) Rozložte: 6x³ − 12x² =                 (6x²(x−2))
```
