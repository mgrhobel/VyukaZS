## PRŮBĚH HODINY

### 1. ÚVOD – Váhy v rovnováze (7 min)

**Vizualizace:**
```
   [x + 5]     =     [12]
   ───────────────────────
      váha vlevo  =  váha vpravo
```

> „Co musím udělat, abych zjistil, co váží x?"
> „Cokoliv udělám na jedné straně vah, musím udělat i na druhé!"

**Klíčový pojem:**
- **Rovnice** = tvrzení o rovnosti dvou výrazů, kde je neznámá (x)
- **Kořen (řešení) rovnice** = hodnota x, při které je rovnost pravdivá
- **Ověření** – dosadíme kořen a zkontrolujeme, zda levá strana = pravá strana

```
Prosté rovnice (ústně):
  x + 5 = 12  →  x = 7    ověření: 7 + 5 = 12  ✓
  3x = 21     →  x = 7    ověření: 3×7 = 21  ✓
  x/4 = 7     →  x = 28   ověření: 28/4 = 7  ✓
```

### 2. VÝKLAD – Rovnice ax + b = c (10 min)

**Ekvivalentní úpravy (mohu je provádět, aniž narušíme rovnost):**
```
1. Přičíst/odečíst stejné číslo na obou stranách
2. Násobit/dělit oba dvě strany stejným NENULOVÝM číslem
```

**Standardní postup:**
```
Krok 1: Přesunout čísla (bez x) doprava  [přičíst/odečíst]
Krok 2: Přesunout x-ové členy doleva     [přičíst/odečíst]
Krok 3: Podělit obě strany koeficientem u x
Krok 4: OVĚŘIT!
```

**Příklad 1:** 5x − 3 = 22
```
5x − 3 = 22
5x = 22 + 3      (+3 na obě strany)
5x = 25
x = 5            (÷5 na obě strany)

Ověření: 5 × 5 − 3 = 25 − 3 = 22  ✓
```

**Příklad 2:** 7 − 2x = 1
```
−2x = 1 − 7
−2x = −6
x = 3            (÷(−2) na obě strany)

Ověření: 7 − 2×3 = 7 − 6 = 1  ✓
```

**Příklad 3 – neznámá na obou stranách:** 4x − 3 = 2x + 7
```
4x − 2x = 7 + 3    (neznámé doleva, čísla doprava)
2x = 10
x = 5

Ověření: 4×5 − 3 = 17;   2×5 + 7 = 17  ✓
```

**Příklad 4 – závorky:** 3(x − 2) = 2(x + 5)
```
3x − 6 = 2x + 10   (rozvinout závorky)
3x − 2x = 10 + 6
x = 16

Ověření: 3(16−2) = 42;   2(16+5) = 42  ✓
```

### 3. FRONTÁLNÍ PROCVIČOVÁNÍ (10 min)

Žáci pracují v sešitech, střídají se u tabule:

| Rovnice | Kořen |
|---|---|
| 3x + 8 = 23 | **x = 5** |
| 2x − 7 = x + 4 | **x = 11** |
| 5(x + 1) = 3(x + 7) | **x = 8** |
| −4x + 12 = −8 | **x = 5** |
| x/3 − 2 = 4 | **x = 18** |

> Každý si ověří (dosadí). Kdo ověří jako první, dá výsledek třídě.

### 4. SKUPINOVÁ PRÁCE – Slovní úlohy (10 min)

Skupiny po 3–4 žácích, každá skupina dostane 2 úlohy (různé typy):

**Skupina A – Geometrická:**
```
Obvod obdélníku je 54 cm. Delší strana je 2× delší než kratší. Jaké jsou délky stran?

Kratší strana: x; delší: 2x
Obvod: 2(x + 2x) = 54 → 6x = 54 → x = 9
Strany: 9 cm a 18 cm    ověření: 2(9+18) = 54  ✓
```

**Skupina B – Věkový:**
```
Otci je 4× více let než synovi. Za 6 let bude otci 2× více.
Kolik jim je teď?

Syn: x; otec: 4x
Za 6 let: (4x + 6) = 2(x + 6) → 4x + 6 = 2x + 12 → 2x = 6 → x = 3
Syn: 3 roky; otec: 12 let    ověření: za 6 let: 18 = 2×9  ✓
```

**Skupina C – Peněžní:**
```
Za 3 sešity a knížku zaplatím 174 Kč. Knížka stojí 3× víc než sešit. Kolik stojí sešit?

Sešit: x; knížka: 3x
3x + 3x = 174 → 6x = 174 → x = 29 Kč
Sešit 29 Kč, knížka 87 Kč   ověření: 3×29 + 87 = 174  ✓
```

**Skupina D – Pohybová:**
```
Dvě auta vyjíždějí vstříc z míst vzdálených 360 km.
První jede 80 km/h, druhé 100 km/h. Za jak dlouho se setkají?

Čas: x hodin
80x + 100x = 360 → 180x = 360 → x = 2 hodiny
```

### 5. ZÁVĚR + EXIT TICKET (8 min)

**Shrnutí (žáci říkají postup):**
1. Rozvinout závorky
2. Neznámé doleva, čísla doprava
3. Podělit koeficientem
4. Vždy ověřit!

**Domácí úkol:** SÚ str. 50–52, č. 1–30

**Exit ticket:**
```
Vyřeš rovnici a ověř výsledek:
  a) 2(x − 3) = x + 7            (x = 13; ověření: 2×10 = 20  ✓)
  b) 5x − 4 = 3x + 10            (x = 7; ověření: 31 = 31  ✓)
```

## DIFERENCIACE

| Skupina | Přístup |
|---|---|
| **Rychlí žáci** | Rovnice s desetinnými koeficienty (0,5x + 1,2 = 3,7); slovní úlohy Skupina D; vlastní slovní úloha vedoucí na rovnici |
| **Průměrní žáci** | Standardní výuka dle přípravy |
| **Žáci s obtížemi** | Pouze rovnice tvaru ax + b = c (bez neznámé na obou stranách); schéma „Neznámé doleva, čísla doprava" stále na tabuli; krok ověření povinný |
