### H1 – Diagnostika + Lomené výrazy: úvod
**Diagnostická písemka:**
```
1. Vyřeš soustavu: 2x+3y=11; x−y=3
2. Rozlož: a) x²−9   b) x²+6x+9   c) 3x²−12x
3. Procentní: Zboží zdražilo o 20 % na 840 Kč. Původní cena?
4. Pravděpodobnost: Hod 2 kostkami. P(součet = 7)?
5. Válec r=5 cm, v=8 cm: povrch a objem.
```

**Lomené výrazy – úvod:**
```
Lomený výraz: a/b kde b je výraz s proměnnou
Podmínka existence: b ≠ 0

(x+2)/(x−3): definováno pro x ≠ 3
(x+1)/(x²−4) = (x+1)/((x+2)(x−2)): def. pro x ≠ 2 a x ≠ −2
```

---

### H2 – Krácení lomených výrazů
```
KLÍČ: Nejdříve ROZLOŽIT čitatel i jmenovatel na součin, pak zkrátit!

(x²−9)/(x+3) = (x+3)(x−3)/(x+3) = x−3  (x ≠ −3)

(6x²−4x)/(2x) = 2x(3x−2)/(2x) = 3x−2  (x ≠ 0)

(x²−x−6)/(x²+x−2):
  Čitatel: x²−x−6 = (x−3)(x+2)
  Jmenovatel: x²+x−2 = (x+1)(x−2)  ← tyto nelze zkrátit → výsledek je zlomek
```

**DÚ:** SÚ str. 5, č. 1–12

### H3 – Rozšiřování lomených výrazů + Sčítání a odčítání
```
Sčítání: převést na společného jmenovatele (NSN výrazů)

1/x + 2/(x+1) = (x+1)/(x(x+1)) + 2x/(x(x+1)) = (x+1+2x)/(x(x+1)) = (3x+1)/(x(x+1))

3/(x−2) − 1/(x+2) = 3(x+2)/((x−2)(x+2)) − (x−2)/((x−2)(x+2))
                   = (3x+6 − x+2) / ((x−2)(x+2))
                   = (2x+8) / (x²−4)
                   = 2(x+4) / (x²−4)
```

**DÚ:** SÚ str. 6, č. 13–24

(a/b) ÷ (c/d) = (a/b) × (d/c) = ad/bc

(x²−4)/(x+1) × (x+1)/(x−2) = (x+2)(x−2)/(x+1) × (x+1)/(x−2) = x+2

## TÝDNY 4–6 – Kvadratické rovnice (18 hodin)

### H7 – Kvadratická rovnice: definice a neúplné typy
```
Kvadratická rovnice: ax² + bx + c = 0  (a ≠ 0)
Typy:
  Neúplná typ 1: ax² = 0  → x = 0
  Neúplná typ 2: ax² + c = 0 → x² = −c/a → x = ±√(−c/a) (nutno −c/a ≥ 0)
  Neúplná typ 3: ax² + bx = 0 → x(ax + b) = 0 → x = 0 nebo x = −b/a
  Úplná: ax² + bx + c = 0 → diskriminant

Příklady neúplných:
3x² − 48 = 0 → x² = 16 → x = ±4
2x² + 3x = 0 → x(2x+3) = 0 → x = 0 nebo x = −3/2
```

**DÚ:** SÚ str. 20, č. 1–10

---

### H8 – Diskriminant a vzorec pro kořeny
```
Diskriminant: D = b² − 4ac

D > 0: dva různé reálné kořeny  x₁,₂ = (−b ± √D) / (2a)
D = 0: jeden dvojnásobný kořen  x₁ = x₂ = −b / (2a)
D < 0: žádný reálný kořen

Příklad: 2x² − 5x + 3 = 0
D = 25 − 4×2×3 = 25 − 24 = 1
x₁ = (5 + 1) / 4 = 6/4 = 3/2
x₂ = (5 − 1) / 4 = 4/4 = 1
Ověření: 2×(3/2)² − 5×(3/2) + 3 = 9/2 − 15/2 + 3 = 0 ✓
```

**DÚ:** SÚ str. 21, č. 11–20

---

### H9 – Cvičení: kvadratické rovnice všech typů
```
10 rovnic stupňující obtížnosti:
1. x² − 9 = 0                    → x = ±3
2. 3x² + 12 = 0                  → D < 0, žádné řešení
3. 5x² − 3x = 0                  → x = 0 nebo x = 3/5
4. x² − 5x + 6 = 0               → (x−2)(x−3)=0 → x=2, x=3
5. 2x² + 7x + 3 = 0              → D = 49−24 = 25; x=−1/2, x=−3
6. x² − 4x + 4 = 0               → D = 0; x₁=x₂=2
7. 3x² − 2x + 5 = 0              → D = 4−60 < 0; ∅
8. (x+1)(x−3) = 5                → x²−2x−3 = 5 → x²−2x−8 = 0 → x=4, x=−2
9. x/2 + 3/x = 2 (× 2x)         → x² + 6 = 4x → x²−4x+6=0 → D<0 → ∅
10. (x−2)² = 3x − 2              → x²−4x+4 = 3x−2 → x²−7x+6=0 → x=1, x=6
```

### H10–H11 – Viètovy vzorce a slovní úlohy

**H10 – Viètovy vzorce:**
```
Pokud x₁, x₂ jsou kořeny ax² + bx + c = 0, pak:
x₁ + x₂ = −b/a    (součet kořenů)
x₁ × x₂ = c/a     (součin kořenů)

Příklad: x² − 5x + 6 = 0 → x₁+x₂ = 5; x₁×x₂ = 6 → x₁=2, x₂=3 (bez počítání!)

Využití: ověřit výsledky; naopak sestavit rovnici z kořenů:
Kořeny 3 a −4 → (x−3)(x+4) = 0 → x² + x − 12 = 0
```

**H11 – Slovní úlohy vedoucí na kvadratické rovnice:**
```
Typ 1 – Geometrický: Obdélník má obvod 36 cm. Obsah 80 cm². Rozměry?
  2(a+b) = 36 → a+b = 18 → b = 18−a
  a × (18−a) = 80 → 18a − a² = 80 → a² − 18a + 80 = 0
  D = 324 − 320 = 4; a = (18±2)/2 → a=10 nebo a=8 → strany 10 cm a 8 cm ✓

Typ 2 – Pohybový: Vlak ujel 240 km. Kdybychom zvýšili rychlost o 20 km/h, 
                  trvalo by to o 1 hodinu méně. Původní rychlost?
  t = 240/v; t−1 = 240/(v+20) → 240/v − 240/(v+20) = 1
  → rovnice vede na v² + 20v − 4800 = 0 → v = 60 km/h
```

## TÝDNY 7–8 – Kvadratická funkce (10 hodin)

### H13 – Funkce y = ax²
```
a > 0: parabola otevřená nahoru (∪); vrchol V = [0, 0]
a < 0: parabola otevřená dolů (∩)
|a| větší: parabola užší; |a| menší: parabola širší

Tabulka pro y = x²:
x: −3, −2, −1, 0, 1, 2, 3
y:  9,  4,  1, 0, 1, 4, 9
```

---

### H14 – Funkce y = a(x − m)² + n  (vrcholový tvar)
```
Vrchol: V = [m, n]; osa souměrnosti: x = m

y = 2(x − 3)² + 1: V = [3, 1]; osa: x = 3; otevřena nahoru (a=2>0)
y = −(x + 2)² + 5: V = [−2, 5]; osa: x = −2; otevřena dolů (a=−1<0)
```

---

### H15 – Obecný tvar ↔ vrcholový tvar
```
y = x² − 6x + 5 → doplnění na čtverec:
  y = (x² − 6x + 9) − 9 + 5
  y = (x − 3)² − 4  → V = [3, −4]

Alternativně: m = −b/(2a) = 6/2 = 3; n = f(3) = 9−18+5 = −4 → V = [3,−4]
```

---

### H16 – Průsečíky paraboly s osami + vlastnosti
```
Průsečík s osou y: x=0 → y = c → bod [0, c]
Průsečíky s osou x: y=0 → ax²+bx+c=0 → diskriminant → nulové body

y = x² − 4x + 3:
  S osou y: [0, 3]
  S osou x: x²−4x+3=0 → (x−1)(x−3)=0 → [1,0] a [3,0]
  Vrchol: m=2, n=−1 → V=[2,−1]
```

## TÝDNY 9–10 – Podobnost trojúhelníků (12 hodin)

### H17 – Podobnost: definice a koeficient
```
Dva trojúhelníky jsou podobné (△ABC ~ △DEF), pokud:
1. Jejich úhly jsou shodné (stejné úhly)
2. Poměry odpovídajících stran jsou stejné (koeficient podobnosti k)

k = DE/AB = EF/BC = DF/AC
```

### H18 – Věty o podobnosti
```
Věty (analogie ke shodnosti, ale méně dat potřeba):
ss: poměry dvou stran jsou stejné A jimi sevřený úhel je shodný
uu: dva úhly jsou shodné (třetí je pak automaticky také)
sss: poměry všech tří stran jsou stejné

Pozor: uu stačí! (třetí úhel je dán jako 180° − α − β)
```

### H19–H20 – Výpočty ze podobnosti
```
△ABC ~ △DEF, k = 3/2
AB = 6 cm → DE = 6 × 3/2 = 9 cm
Obsah: Poměr obsahů = k² = 9/4

Praktická úloha: Výška stromu pomocí stínu.
Stín stromu = 8 m; stín tyče 2 m (výška tyče 1,5 m)
Výška stromu / 1,5 = 8 / 2 → výška = 6 m
```

### H21 – Podobnost a Thaletova věta
```
Thaletova věta: Přímky rovnoběžné s podstavou trojúhelníku dělí boční strany v shodném poměru.

Využití: výpočet neznámé délky z rovnoběžných řezů
```

## TÝDNY 11–12 – Trigonometrie pravoúhlého trojúhelníku (12 hodin)

### H22 – Sin, cos, tan: definice
```
V pravoúhlém trojúhelníku (pravý úhel u vrcholu C):
  sin α = protilehlá odvěsna / přepona = a / c
  cos α = přilehlá odvěsna / přepona  = b / c
  tan α = protilehlá / přilehlá       = a / b

Mnemotechnika: SOH-CAH-TOA (anglicky)
  Sin = Opposite/Hypotenuse
  Cos = Adjacent/Hypotenuse  
  Tan = Opposite/Adjacent
```

### H23 – Výpočty stran z úhlu
```
Příklad: α = 35°, přepona c = 10 cm. Najdi a a b.
  sin 35° = a/10 → a = 10 × sin 35° ≈ 10 × 0,574 = 5,74 cm
  cos 35° = b/10 → b = 10 × cos 35° ≈ 10 × 0,819 = 8,19 cm

Tabulky nebo kalkulačka (v JPZ jsou tabulky povoleny!)
```

### H24 – Výpočty úhlů ze stran
```
Příklad: a = 6 cm, c = 10 cm. Najdi α.
  sin α = 6/10 = 0,6 → α = arcsin(0,6) ≈ 36,87° ≈ 37°

Goniometrická tabulka: hledáme hodnotu 0,6 ve sloupci sin → odečteme úhel
```

### H25–H26 – Aplikace trigonometrie
```
Úloha 1 (výška objektu): Pozorovatel ve vzdálenosti 50 m od budovy vidí střechu pod 
  úhlem 40° nad horizontem (a očima 1,7 m nad zemí).
  tan 40° = výška_nad_očima / 50
  výška_nad_očima = 50 × tan 40° ≈ 42 m → celková výška ≈ 43,7 m

Úloha 2 (sklon svahu): Silnice překonává výškový rozdíl 120 m na délce 900 m.
  sin α = 120/900 = 2/15 → α ≈ 7,7°
```

### H27 – Písemka č. 3 – Trigonometrie + Podobnost

### H28–H30 – Pravděpodobnost a kombinatorika
```
Pravidlo součtu: A nebo B (disjunktní): |A∪B| = |A| + |B|
Pravidlo součinu: A a pak B: |A×B| = |A| × |B|

Permutace n prvků: n! = n×(n−1)×…×1  (pořadí záleží, všechny prvky)
  Kolik slov lze sestavit ze slova KOLO? 4!/2! = 12 (2 stejná O)

Kombinace C(n,k) = n! / (k!(n−k)!)  (výběr k z n, pořadí nezáleží)
  Kolik 3-členných skupin z 8 žáků? C(8,3) = 56
```
