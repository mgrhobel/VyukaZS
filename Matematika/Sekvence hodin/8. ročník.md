### H2–H4 – Mocniny: pravidla a výpočty

**H2 – Pravidla pro mocniny:**
```
a^m × a^n = a^(m+n)       →  2^3 × 2^4 = 2^7 = 128
a^m ÷ a^n = a^(m−n)       →  3^5 ÷ 3^2 = 3^3 = 27
(a^m)^n = a^(m×n)         →  (2^3)^2 = 2^6 = 64
(a × b)^n = a^n × b^n     →  (2 × 3)^4 = 2^4 × 3^4 = 16 × 81 = 1 296
(a / b)^n = a^n / b^n     →  (2/3)^3 = 8/27
```

**H3 – Záporný základ mocniny:**
```
(−3)^2 = 9      (sudý mocnitel → vždy kladný)
(−3)^3 = −27    (lichý mocnitel → záporný)

POZOR na závorky:
(−3)^2 = 9    ale    −3^2 = −(3^2) = −9   (různé výsledky!)
```

**H4 – Procvičení + smíšené příklady:**
```
Zjednodušši: (2x)^3 × (2x)^2 = (2x)^5
Vypočítej: (−2)^3 + (−3)^2 × 2 − (−1)^10 = −8 + 18 − 1 = 9
```

## TÝDNY 3–4 – Odmocniny (10 hodin)

### H5 – Druhá odmocnina
```
√a je číslo x ≥ 0, pro které platí x² = a
(Podmínka: a ≥ 0; √(−4) neexistuje v ℝ)

√0 = 0;  √1 = 1;  √4 = 2;  √9 = 3;  √16 = 4;  √25 = 5
√36 = 6;  √49 = 7;  √64 = 8;  √81 = 9;  √100 = 10

Přibližně (kalkulačkou): √2 ≈ 1,414;  √3 ≈ 1,732;  √5 ≈ 2,236
```

---

### H6 – Třetí odmocnina + Pravidla pro odmocniny
```
∛a = číslo x, pro které x³ = a  (existuje i pro záporná a!)
∛8 = 2;  ∛27 = 3;  ∛64 = 4;  ∛(−8) = −2

Pravidla:
√(a × b) = √a × √b          →   √(4 × 9) = 2 × 3 = 6
√(a / b) = √a / √b          →   √(16/25) = 4/5
√(a²) = |a|                 →   √(5²) = 5;  √((−3)²) = 3
```

### H7–H8 – Pythagorova věta s odmocninami + vzdálenost
```
Vzdálenost dvou bodů v souřadnicové soustavě:
d(A,B) = √((x₂−x₁)² + (y₂−y₁)²)

Příklad: A=[1,2], B=[5,5]
d = √((5−1)² + (5−2)²) = √(16 + 9) = √25 = 5
```

## TÝDNY 5–7 – Lineární funkce (14 hodin)

### H9 – Co je funkce
```
Funkce: každému x z definičního oboru přiřadí právě jedno y
Zápis: y = f(x) nebo f: x → y

Způsoby zadání:
1. Vzorcem:  y = 2x + 3
2. Tabulkou: x: 0, 1, 2, 3 → y: 3, 5, 7, 9
3. Grafem:   body v souřadnicové soustavě
```

---

### H10 – Přímá úměra jako funkce + souřadnicová soustava
```
y = kx  (přímá úměra, přechází počátkem)

Graf: přímka procházející [0,0]
k > 0 → rostoucí (jde doleva doprava nahoru)
k < 0 → klesající
k = 0 → přímka splývá s osou x

Příklady:
y = 3x: body [0,0], [1,3], [2,6], [−1,−3]
y = −2x: body [0,0], [1,−2], [2,−4]
```

---

### H11 – Lineární funkce y = kx + q
```
k = směrnice (sklon přímky)
q = průsečík s osou y (kde přímka protíná osu y)

y = 2x + 3:
  x = 0: y = 3  → bod [0, 3]  (= q)
  x = 1: y = 5  → bod [1, 5]
  Přímka stoupá (k = 2 > 0)

y = −x + 4:
  x = 0: y = 4  → bod [0, 4]
  x = 4: y = 0  → bod [4, 0]  (= průsečík s osou x)
  Přímka klesá (k = −1 < 0)
```

**DÚ:** SÚ str. 40, č. 1–10

---

### H12 – Jak sestrojit graf lineární funkce
```
Metoda 1: Dvěma body
  y = 3x − 6: vezmi x=0 → y=−6; x=2 → y=0. Spoj body [0,−6] a [2,0].

Metoda 2: Přes k a q
  y = 2x + 1: začni v [0,1], pak "2 doprava, 2 nahoru" (k=2 → Δy/Δx = 2/1)
```

---

### H13 – Vlastnosti lineárních funkcí + srovnání
```
Rostoucí: k > 0    Klesající: k < 0    Konstantní: k = 0
Rovnoběžné přímky: stejné k, různé q
Souhlasné: stejné k, stejné q (totožné!)
Průsečíky: dvě přímky se protínají, kde mají stejné y (→ řešit soustavu)

Graf y = 2x + 1 a y = 2x − 3: rovnoběžné (k = 2 u obou)
Graf y = 3x − 2 a y = −x + 6: průsečík? 3x−2 = −x+6 → 4x=8 → x=2, y=4 → [2,4]
```

---

### H14–H15 – Průsečíky a aplikace lineárních funkcí
**H14 – Průsečíky přímky s osami:**
```
Průsečík s osou y: x=0 → P_y = [0, q]
Průsečík s osou x: y=0 → 0 = kx+q → x = −q/k → P_x = [−q/k, 0]

y = 4x − 8: P_y = [0,−8]; P_x: 0=4x−8 → x=2 → [2,0]
```

**H15 – Aplikace: ceník, tarif, pohyb:**
```
Taxi: základní poplatek 50 Kč + 22 Kč/km → y = 22x + 50 (x = km)
Porovnání dvou tarifů: kdy se vyplatí přejít? (průsečík grafů)
```

**DÚ H15:** SÚ str. 43, č. 15–22

## TÝDNY 8–9 – Lineární rovnice a nerovnice (12 hodin)

### H17 – Rovnice s lomenými výrazy
```
Postup: Vynásobit celou rovnici LCM (nejmenší společný násobek) jmenovatelů

x/3 + x/4 = 7    | × 12 (NSN(3,4) = 12)
4x + 3x = 84
7x = 84
x = 12

Ověření: 12/3 + 12/4 = 4 + 3 = 7 ✓
```

**DÚ:** SÚ str. 55, č. 1–10

---

### H18 – Lineární nerovnice
```
Pravidlo: při násobení/dělení záporným číslem se obrátí nerovnítko!

3x − 2 > 7     →  3x > 9   →  x > 3   (graficky: otevřená část osy vpravo od 3)
−2x + 4 ≤ 10  →  −2x ≤ 6  →  x ≥ −3  (dělení záporným: obrátit!)

Zápis množiny řešení:
  x > 3: (3; +∞)   nebo na číselné ose: ○ –– –– → (o = otevřený bod)
  x ≥ 3: [3; +∞)   nebo na číselné ose: ● –– –– → (● = uzavřený bod)
```

**DÚ:** SÚ str. 57, č. 15–24

---

### H19 – Slovní úlohy vedoucí na nerovnice
```
„Jana ušetřila 450 Kč. Chce koupit knížky za 85 Kč a zbýt jí musí aspoň 100 Kč."
85x ≤ 450 − 100 = 350  →  x ≤ 4,1  →  může koupit max. 4 knížky
```

## TÝDNY 10–12 – Soustava dvou lineárních rovnic (14 hodin)

### H20 – Soustava rovnic: úvod + grafická metoda
```
Soustava: 2 rovnice o 2 neznámých; hledáme x, y splňující obě najednou

Grafická metoda: nakreslíme 2 přímky; průsečík = řešení
  y = 2x + 1
  y = −x + 4

2x + 1 = −x + 4 → 3x = 3 → x = 1; y = 3 → řešení: (1, 3)
```

---

### H21 – Metoda sčítací (eliminační)
```
Postup: Upravit rovnice tak, aby po sečtení zmizela jedna neznámá

2x + 3y = 11   | × 2   →   4x + 6y = 22
4x − y = 5                  4x − y = 5
                           ──────────────
                           Odečteme: 7y = 17 → y = 17/7...

Lepší příklad (celé výsledky):
3x + 2y = 12
x − 2y = 4    (sečíst přímo)
────────────
4x = 16 → x = 4; dosadíme: 4 − 2y = 4 → y = 0
```

**DÚ:** SÚ str. 65, č. 1–10

---

### H22 – Metoda dosazovací (substituční)
```
Postup: Vyjádřit jednu neznámou z jedné rovnice, dosadit do druhé

y = 3x − 1      (z první rovnice)
2x + y = 9      → 2x + (3x − 1) = 9 → 5x = 10 → x = 2; y = 5
```

**DÚ:** SÚ str. 66, č. 11–20

---

### H23–H25 – Slovní úlohy na soustavu rovnic
```
Typ 1 – Věkový: Součet věků otce a syna je 50. Otec je o 26 starší. Kolik je každému?
  x + y = 50
  x − y = 26     → sečíst: 2x = 76 → x = 38, y = 12

Typ 2 – Cena: Lístek pro dospělého stojí a Kč, pro dítě d Kč.
  2 dospělí + 3 děti = 430 Kč
  1 dospělý + 4 děti = 360 Kč
  2a + 3d = 430
  a + 4d = 360   → soustava, řeš

Typ 3 – Pohyb: Dva cyklisté jedou vstříc, vzdálenost 84 km, rychlosti v₁, v₂.
  Za 2 hodiny se setkají: 2v₁ + 2v₂ = 84 → v₁ + v₂ = 42
  v₁ = v₂ + 6  (jeden je o 6 km/h rychlejší) → soustava
```

```
1. Vyřeš sčítací metodou: 3x + 2y = 16; x + 2y = 8
2. Vyřeš dosazovací metodou: y = 2x − 3; 3x + y = 12

```

## TÝDNY 13–14 – Mnohočleny a algebraické výrazy (12 hodin)

### H27 – Mnohočlen: definice a operace
```
Monomon: ax^n (jeden člen); Binomon: ax^n + bx^m; Trinomon: 3 členy; ...

Sčítání/odčítání: sčítat POUZE stejnorodé členy (stejná proměnná, stejný mocnitel)
  (3x² + 2x − 5) + (x² − 4x + 3) = 4x² − 2x − 2

Záporka před závorkou = otočit všechna znaménka:
  − (2x − 3) = −2x + 3
```

---

### H28 – Násobení mnohočlenů
```
Monon × mnohočlen: 2x(3x² − x + 4) = 6x³ − 2x² + 8x

Mnohočlen × mnohočlen: FOIL nebo každý s každým:
  (x + 3)(x − 5) = x² − 5x + 3x − 15 = x² − 2x − 15
  (2x + 1)(3x − 4) = 6x² − 8x + 3x − 4 = 6x² − 5x − 4
```

---

### H29 – Algebraické vzorce
```
(a + b)² = a² + 2ab + b²
(a − b)² = a² − 2ab + b²
(a + b)(a − b) = a² − b²    ← "rozdíl čtverců"

Příklady:
(x + 5)² = x² + 10x + 25
(2x − 3)² = 4x² − 12x + 9
(3x + 2)(3x − 2) = 9x² − 4

Opačně (rozklad!):
x² − 16 = (x + 4)(x − 4)
x² + 6x + 9 = (x + 3)²
```

---

### H30 – Rozklad mnohočlenu na součin
```
1. Vytýkání: 6x² + 9x = 3x(2x + 3)
2. Pomocí vzorců: x² − 25 = (x+5)(x−5); 4x² + 4x + 1 = (2x+1)²
3. Kombinace: 2x² − 8 = 2(x² − 4) = 2(x+2)(x−2)
```

**DÚ:** SÚ str. 80, č. 1–20

## TÝDNY 15–17 – Geometrie: kružnice, válec, kužel, koule (16 hodin)

### H31–H32 – Kružnice: tečna, vepsaná a opsaná
```
Tečna ke kružnici: dotýká se v 1 bodě, kolmá na poloměr v bodě dotyku
Kružnice vepsaná trojúhelníku: střed = průsečík os úhlů; r = vzdálenost středu od strany
Kružnice opsaná trojúhelníku: střed = průsečík os stran; R = vzdálenost středu od vrcholu
```

### H33–H34 – Válec
```
Válec: dvě kruhové podstavy, boční plášť (rozvinutý: obdélník)
Síť válce: dva kruhy + obdélník (šířka = o = 2πr, výška = v válce)

Povrch: S = 2πr² + 2πrv
Objem:  V = πr²v

Příklad: r = 5 cm, v = 12 cm
S = 2π×25 + 2π×5×12 = 50π + 120π = 170π ≈ 534 cm²
V = π×25×12 = 300π ≈ 942 cm³
```

### H35–H36 – Kužel
```
Kužel: kruhová podstava, hrotová část
Délka tvornice l = √(r² + v²)  (Pythagorova věta!)

Povrch: S = πr² + πrl  (podstava + plášť)
Objem:  V = 1/3 × πr²v

Příklad: r = 6 cm, v = 8 cm
l = √(36 + 64) = √100 = 10 cm
S = π×36 + π×6×10 = 36π + 60π = 96π ≈ 301 cm²
V = 1/3 × π×36×8 = 96π ≈ 301 cm³
```

### H37 – Koule
```
Povrch: S = 4πr²
Objem:  V = 4/3 × πr³

Příklad: r = 3 cm
S = 4π×9 = 36π ≈ 113 cm²
V = 4/3 × π×27 = 36π ≈ 113 cm³  (náhoda stejné v tomto příkladu!)
```

### H38 – Písemka č. 4 – Tělesa
```
1. Válec: r = 4 cm, v = 10 cm. Vypočítej povrch a objem.
2. Kužel: r = 3 cm, v = 4 cm. Vypočítej délku tvornice, povrch a objem.
3. Koule: S = 100π cm². Urči r a V.
4. Kombinovaná: Lopata pro zmrzlinu tvoří polokoule (r = 4 cm) + kužel (v = 6 cm).
                Jaký je objem zmrzliny?
5. Zásobník vody má tvar válce (r = 2 m, v = 5 m). Kolik litrů pojme?
   (1 m³ = 1 000 l)
```

### H40–H41 – Základy pravděpodobnosti
```
Náhodný jev A; jevový prostor Ω (všechny možné výsledky)
P(A) = |A| / |Ω|   (počet příznivých / celkový počet, za předpokladu stejné pravděpodobnosti)

Příklady:
Hod kostkou: Ω = {1,2,3,4,5,6}; P(sudé) = 3/6 = 1/2
Výběr karty z 32karet: P(eso) = 4/32 = 1/8
Hod 2 mincemi: Ω = {OO, OP, PO, PP}; P(aspoň 1 panna) = 3/4

Pravděpodobnost opačného jevu: P(Ā) = 1 − P(A)
```
