## 1. Viètovy vzorce – testové otázky

*(Původně: Matematika 9.r. – Test č. 2, Otázka 3)*

**3. (4 b) Viètovy vzorce:**

a) Rovnice x² + bx + 12 = 0 má kořeny x₁=3, x₂=4. Najdi b.

b) Jeden kořen rovnice x² − 5x + c = 0 je x₁=2. Najdi druhý kořen a c.

c) Sestavte kvadratickou rovnici s kořeny x₁=−2 a x₂=5.

d) Součin kořenů rovnice 2x² − 7x + k = 0 je 3. Jaká je hodnota k?

### Klíč

a) x₁+x₂=−b=7 → **b=−7**; x₁×x₂=12 ✓
b) součet=5 → **x₂=3**; c=x₁×x₂=**6**
c) (x+2)(x−5)=0 → **x²−3x−10=0**
d) c/a=k/2=3 → **k=6**

## 2. Kombinatorika – složená pravděpodobnost

*(Původně: Matematika 8.r. – Závěrečný test č. 5, Otázka 11d)*

**Pravděpodobnost: V sáčku: 4 červené, 3 modré, 3 žluté. Vytáhneme 2 (bez vracení).**

d) P(obě červené)?

### Klíč

```
Příznivé: C(4,2) = 4!/(2!·2!) = 6
Všechny:  C(10,2) = 10!/(2!·8!) = 45
P(obě červené) = 6/45 = 2/15 ≈ 0,133 (13,3 %)
```

## 3. Soustava tří rovnic – bonusová úloha

*(Původně: Matematika 8.r. – Test č. 3, Bonusová otázka 4)*

**4. (2 b) Bonusová:**

Soustava: 2x + 3y + z = 10; x − y = 2; z = 3. Řešte.

### Klíč

```
z = 3
x − y = 2  →  x = y + 2

Dosazení: 2(y+2) + 3y + 3 = 10
          2y + 4 + 3y + 3 = 10
          5y + 7 = 10
          5y = 3
          y = 3/5

x = y + 2 = 3/5 + 10/5 = 13/5

Řešení: x = 13/5; y = 3/5; z = 3
```
