# SŠ Matematika – Viètovy vzorce

> **Úroveň:** Střední škola (1. ročník gymnázia / SOŠ)  
> **Navazuje na:** Kvadratické rovnice (diskriminant) – ZŠ 9. ročník  
> **RVP:** Není součástí RVP ZV 2023; zařazeno v RVP G (gymnázia)

---

## Co jsou Viètovy vzorce?

Jsou-li x₁ a x₂ kořeny kvadratické rovnice **ax² + bx + c = 0** (a ≠ 0), platí:

```
x₁ + x₂ = −b/a       (součet kořenů)
x₁ × x₂ = c/a        (součin kořenů)
```

Vzorce pojmenované po francouzském matematikovi **Françoisu Vièteovi** (1540–1603).

---

## Odvození

Kořeny jsou dány vzorcem: x₁,₂ = (−b ± √D) / 2a

```
x₁ + x₂ = (−b + √D)/2a + (−b − √D)/2a = −2b/2a = −b/a  ✓
x₁ × x₂ = (−b + √D)(−b − √D) / 4a²
         = (b² − D) / 4a²
         = (b² − (b²−4ac)) / 4a²
         = 4ac / 4a²
         = c/a  ✓
```

## Příklady

### Příklad 1 – Ověření kořenů
Rovnice: x² − 5x + 6 = 0

Diskriminantem: x=2, x=3

Viètovy vzorce: x₁+x₂ = 5 = 5/1 ✓  ;  x₁×x₂ = 6 = 6/1 ✓

---

### Příklad 2 – Nalezení koeficientu
Rovnice x² + bx + 12 = 0 má kořeny x₁=3, x₂=4. Najdi b.

```
x₁ + x₂ = −b/1 = 7  →  b = −7
x₁ × x₂ = 12/1 = 12  ✓  (ověření: 3×4=12 ✓)
Rovnice: x² − 7x + 12 = 0
```

---

### Příklad 3 – Druhý kořen a koeficient c
Rovnice x² − 5x + c = 0 má kořen x₁=2. Najdi x₂ a c.

```
x₁ + x₂ = 5  →  x₂ = 5 − 2 = 3
x₁ × x₂ = c  →  c = 2 × 3 = 6
Rovnice: x² − 5x + 6 = 0  (ověřit diskriminantem ✓)
```

---

### Příklad 4 – Sestavení rovnice z kořenů
Kořeny jsou x₁ = −2 a x₂ = 5. Napiš kvadratickou rovnici.

```
Součet: −2 + 5 = 3 = −b/1  →  b = −3
Součin: −2 × 5 = −10 = c/1  →  c = −10
Rovnice: x² − 3x − 10 = 0

Alternativní postup: (x − (−2))(x − 5) = (x+2)(x−5) = x² − 3x − 10 ✓
```

---

### Příklad 5 – Součin kořenů s koeficientem a
Rovnice 2x² − 7x + k = 0 má součin kořenů = 3. Najdi k.

```
x₁ × x₂ = c/a = k/2 = 3  →  k = 6
```

---

## Cvičení

1. Kořeny rovnice jsou x₁=−1, x₂=4. Napiš rovnici.
2. Rovnice x² + px + 8 = 0 má kořeny x₁=2. Najdi x₂ a p.
3. Rovnice 3x² − 12x + q = 0 má součin kořenů = 4. Najdi q.
4. Ověř Viètovými vzorci: x²−x−6=0 (kořeny 3 a −2).
5. Koeficient b rovnice x²+bx+5=0, jsou-li kořeny x₁ a x₂ taková, že x₁+x₂=−4.

### Řešení cvičení
1. (x+1)(x−4)=0 → x²−3x−4=0
2. x₂=4, p=−(2+4)=−6 → x²−6x+8=0
3. q/3=4 → q=12
4. x₁+x₂=1=1/1 ✓; x₁×x₂=−6=−6/1 ✓
5. −b=−4 → b=4

## Proč je toto téma pro SŠ?

- RVP ZV 2023 vyžaduje: diskriminant, dva kořeny pomocí vzorce, ověření dosazením
- Viètovy vzorce jsou **abstraktní vztah** vyžadující algebraické myšlení SŠ úrovně
- V JPZ (přijímací zkoušky ZŠ → SŠ) se Viètovy vzorce **neobjevují**
- Zařazeny v RVP G (gymnázia, 1. ročník) a maturitní témata
