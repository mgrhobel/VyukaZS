## ⚠️ Zásadní rozdíl ZŠ vs. SŠ v Informatice

| Téma | ZŠ (RVP ZV 2023) | SŠ (SOŠ / Gymnázium) |
|------|-------------------|-----------------------|
| Programování | Algoritmické myšlení, Scratch, základy Pythonu (if/for/funkce) | Python pokročile, OOP, framework, API |
| Data | Čtení grafů, základní tabulky v Excelu, interpretace | Databáze (SQL pokročile), datová věda, pandas |
| Sítě a web | Princip internetu, bezpečnost, HTML základy | Protokoly, serverový web, správa sítě |
| Databáze | Pojem tabulky, jednoduchý SELECT jako ukázka | Návrh databáze (ER diagram), JOIN, indexy, SQLite |
| Bezpečnost | Phishing, silná hesla, digitální stopa | Kybernetická bezpečnost, šifrování, GDPR v praxi |
| AI | Co je ML, ukázka (Teachable Machine) | Trénink modelu, datové pipeline |
