### 🔑 Klíč – Test mediální gramotnost

**1.** SQL injection = útočník vloží SQL kód do vstupního pole. Obrana: parametrizované dotazy: `cursor.execute("SELECT ... WHERE name = ?", (vstup,))`  
**2.** Phishing: neočekávaná výzva, žádá heslo/osobní data, podezřelá adresa odesílatele, gramatické chyby, podezřelý odkaz (hover odhalí cíl)  
**3.** Symetrické: stejný klíč pro šifrování i dešifrování (rychlé). Asymetrické: veřejný + soukromý klíč. HTTPS = asymetrické pro výměnu klíče, pak symetrické pro přenos dat.  
**4.** Filter bubble = algoritmus doporučuje obsah podobný tomu, co jsme viděli → vidíme jen část reality, extremizace názorů  
**5.** S = Stop (zastav se, nereaguj hned), I = Investigate the source (kdo to napsal), F = Find better coverage (hledej jiné zdroje), T = Trace claims (vystopuj původ tvrzení)  
**6.** ML = počítač se učí z příkladů (dat) místo pevně naprogramovaných pravidel. Příklady: spam filtr, rozpoznávání obličeje, doporučování filmů  
**7.**
```sql
SELECT autor, text FROM zpravy
WHERE overeno = 0
ORDER BY datum DESC;
```
**8.**
```python
def je_bezpecna(url):
    if not url.startswith("https://"):
        return False
    domena = url.replace("https://", "").split("/")[0]
    return "." in domena

print(je_bezpecna("https://www.google.com"))  # True
print(je_bezpecna("http://gmail.com"))         # False
```

## Projekt č. 2 – 3D modelování a 3D tisk
**Termín odevzdání:** Červen (závěr roku)

**Volitelné téma:** Pouze pro třídy/žáky, kteří zvolí Option E v H55–H56

---

### Zadání pro žáky

Navrhni a zdokumentuj 3D model, který bude použitelný v praxi.

**Software (dle výběru):**
- **Tinkercad** (online, zdarma, doporučeno pro začátečníky) – tinkercad.com
- **Blender** (zdarma, pokročilý) – blender.org
- **FreeCAD** (zdarma, parametrické modelování)

**Povinné součásti:**

1. **3D model** – navrhni objekt splňující aspoň jedno z:
   - Přívěsek nebo jmenovka (výrobitelné na 3D tiskárně)
   - Náhradní díl nebo organizer (praktická funkce)
   - Umělecký objekt / skulptura (estetický záměr)

2. **Technické parametry:**
   - Model musí být **manifold** (bez otevřených děr) – aby byl tisknutelný
   - Export do formátu **.STL**
   - Rozměry: max. 10 × 10 × 10 cm (aby byl tisk realistický)

3. **Průvodní dokumentace:**
   - Screenshot modelu z aspoň 2 pohledů (přední a boční)
   - Popis záměru (100 slov): co jsi navrhoval/a, pro koho, proč takový tvar
   - Kdyby byl dostupný 3D tisk: jaký materiál by byl vhodný? (PLA, ABS, PETG – krátce vysvětlit)

**Bonus:** Pokud má škola 3D tiskárnu – žák si může vytisknout model za splnění rozměrových limitů (+2 b navíc)
