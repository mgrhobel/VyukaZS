### ČÁST A – Python (12 b)

**1. (3 b)** Co vypíše tento kód? Napiš přesný výstup.

```python
def pocitej(seznam):
    soucet = 0
    for prvek in seznam:
        if prvek > 0:
            soucet += prvek
    return soucet

cisla = [-3, 5, -1, 8, 2, -7]
print(pocitej(cisla))
```

### 🔑 Klíč – Závěrečný test

**1.** Funkce sečte kladná čísla: 5+8+2 = **15**  
**2.**
```python
def obratSlovo(slovo):
    vysledek = ""
    for i in range(len(slovo)-1, -1, -1):
        vysledek += slovo[i]
    return vysledek
```
**3.**
```python
soucet = 0
pocet = 0
cislo = int(input("Číslo (0=konec): "))
while cislo != 0:
    soucet += cislo
    pocet += 1
    cislo = int(input("Číslo (0=konec): "))
print("Součet:", soucet, "Počet:", pocet)
```
**4.** Ověření pomocí `if` + `.isdigit()`:
```python
pocet_str = input("Kolik žáků? ")
if not pocet_str.isdigit():

else:
    pocet = int(pocet_str)
    for i in range(pocet):
        print(f"Žák {i+1}")
```
