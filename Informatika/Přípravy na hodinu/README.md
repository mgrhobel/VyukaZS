## Co patří kde

### ✅ ZŠ (Základní škola) – RVP ZV 2023
Obsah, který RVP ZV 2023 explicitně uvádí pro 2. stupeň ZŠ:
- Algoritmizace a blokové programování (Scratch)
- Vývojové diagramy a pseudokód
- Textové kódování – úvod do Pythonu (základní funkce)
- Binární soustava a kódování dat
- Digitální gramotnost: bezpečnost, soubory, cloud, hardware
- Tabulkový procesor (Excel), prezentace, textový editor
- Datová gramotnost a interpretace dat
- Networking – princip internetu (konceptuálně)
- Mediální gramotnost (fotografování, postprodukce)
- 3D modelování jako kreativní vyjádření

### 🔴 SŠ (Střední škola) – RVP SOŠ Informatika / Gymnázia
Obsah, který standardně začíná na SŠ:
- SQL (SELECT, JOIN, CREATE TABLE)
- Návrh relační databáze a ER diagramy
- Python pokročile: soubory, výjimky, OOP, moduly
- Síťové nástroje: Wireshark, tracert, konfigurace
- Analýza informačních systémů a jejich architektura
- Webový vývoj (HTML/CSS/JS systematicky)
