### LISTOPAD–PROSINEC (cca 14 hodin)

**Celek 2 – Python pokročile** (dokončení, 8 h)
- Soubory: čtení (`open()`, `f.read()`, iterace po řádcích) a zápis (`"w"`, `"a"`)
- Klíčové slovo `with`: automatické zavření souboru
- Zpracování chyb: `try`, `except ValueError`, `except ZeroDivisionError`, `finally`
- Proč ošetřovat chyby: robustní programy vs. crashing programs
- Projekt: Deník – záznamy ukládány do souboru, načteny při startu

**Celek 3 – Úvod do OOP** (6 h)
- Třída (`class`) a objekt (instance): analogie „plán domu" vs. „postavený dům"
- `__init__`, atributy instance (`self.jmeno`)
- Metody: funkce patřící třídě
- Dědičnost: podtřída rozšiřuje rodičovskou třídu
- Projekt: Třída `Ziak` s atributy a metodami

```python
# Ukázka: hádání čísla s ošetřením chyb
import random
tajne = random.randint(1, 100)
hadani = 0
while hadani != tajne:
    try:
        hadani = int(input("Hádej číslo (1-100): "))
    except ValueError:
        print("Zadej celé číslo!")
        continue
    if hadani < tajne:
        print("Víc!")
    elif hadani > tajne:
        print("Míň!")
print("Správně! 🎉")
```

## Hodnocení v informatice

- **Praktické úkoly** (projekty, úlohy): klíčový způsob hodnocení
- **Funkčnost > estetika:** program, který funguje správně, je lepší než hezky vypadající, ale rozbité řešení
