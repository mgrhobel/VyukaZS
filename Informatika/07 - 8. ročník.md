### LISTOPAD–PROSINEC (cca 14 hodin)

**Celek 2 – Python pokročile** (dokončení, 8 h)
- Soubory: čtení (`open()`, `f.read()`, iterace po řádcích) a zápis (`"w"`, `"a"`)
- Klíčové slovo `with`: automatické zavření souboru
- Zpracování chyb: `try`, `except ValueError`, `except ZeroDivisionError`, `finally`
- Proč ošetřovat chyby: robustní programy vs. crashing programs
- Projekt: Deník – záznamy ukládány do souboru, načteny při startu

**Celek 3 – Turtle grafika a kryptografie** (6 h)
- Turtle graphics rozšíření: spirály, hvězdy, animace, fraktály (strom, vločka)
- Kryptografie: Caesarova šifra – šifrování a dešifrování v Pythonu
- Vigenèrova šifra: klíč, modulární aritmetika
- Lámání šifrovaných zpráv: brute-force (zkoušení všech posunů)
- Projekt: Žáci si navzájem posílají zašifrované zprávy a lámají je

```python
# Ukázka: hádání čísla s ošetřením chyb
import random
tajne = random.randint(1, 100)
hadani = 0
while hadani != tajne:
    try:
        hadani = int(input("Hádej číslo (1-100): "))
    except ValueError:
        print("Zadej celé číslo!")
        continue
    if hadani < tajne:
        print("Víc!")
    elif hadani > tajne:
        print("Míň!")
print("Správně! 🎉")
```
