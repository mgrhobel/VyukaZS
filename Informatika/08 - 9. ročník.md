## Měsíční rozvrh (počítáme s 1–2 h/týden)

### ZÁŘÍ–ŘÍJEN (cca 14 hodin)

**Celek 1 – Python pokročile** (10 h)
- Vstupní diagnostika: OOP, soubory, SQL z 8. ročníku
- Knihovny třetích stran: `pip install requests matplotlib`
- HTTP API: stažení dat z webu, zpracování JSON odpovědi
- Vizualizace dat (matplotlib):
  - Spojnicový graf, sloupcový graf, bodový diagram
  - Popis os, nadpis, mřížka
- Web scraping základy: `requests` + `BeautifulSoup`
  - Etika: robots.txt, nezatěžovat servery, copyright

```python
import requests
odpoved = requests.get(
    "https://api.open-meteo.com/v1/forecast"
    "?latitude=50.07&longitude=14.42&current_weather=true"
)
data = odpoved.json()
print("Teplota v Praze:", data["current_weather"]["temperature"], "°C")
```

**Projekt: Datový průzkum** (4 h)
- Žáci stáhnou reálná data (počasí, kurzy, sporty) via API nebo CSV
- Zpracují statistiky v Pythonu, vizualizují (matplotlib)
- Prezentace 5 min: grafy na obrazovce, interpretace dat

### LISTOPAD–PROSINEC (cca 14 hodin)

**Celek 2 – Závěrečný projekt: zahájení** (8 h)
- Zadání a brainstorming: volné téma, musí řešit reálný problém
- Příklady úspěšných projektů:
  - Aplikace na správu kapesného (Python + soubory)
  - Web s recenzemi knih (HTML + CSS + JS)
  - Bot na analýzu sportovních výsledků (Python + API)
  - Automatizovaný deník počasí (Python + matplotlib)
- Postup: Výběr tématu → schválení → analýza + návrh → kódování

**Celek 4 – Informační společnost a AI** (4 h)
- Co je strojové učení: trénink na datech vs. naprogramovaná pravidla
- Demo: teachablemachine.withgoogle.com (trénink rozpoznávače gest)
- Bias v datech → bias v AI
- Autorská práva na kód: open source, GPL, MIT licence
- Kariéry v IT: programátor, datový analytik, UX designér, kybernetická bezpečnost

**Opakování a příprava na pololetí** (2 h)
- Písemka: databáze, JavaScript, SQL SELECT + JOIN

### LEDEN–ÚNOR (cca 14 hodin)

**Celek 2 – Závěrečný projekt: vývoj** (12 h)
- H40: Analýza + návrh (schéma, pseudokód, ER diagram pokud DB)
- H41–H44: Kódování – učitel konzultuje, poskytuje zpětnou vazbu

**Celek 3 – Kybernetická bezpečnost** (2 h)
- Typy útoků: phishing, malware, ransomware, social engineering
