### BLOK 2 – Úvod do textového programování (12 hodin)

#### H36 – Proč Python? Přechod od Scratch
```
Blokové programování → textové programování
  Scratch: zelená vlajka → Python: spustit soubor
  Scratch: pohyb 10 → Python: příkaz v shellu

Thonny IDE: instalace a orientace
  Shell (interaktivní) vs. editor (soubor .py)
  print("Ahoj světe!")  ← první program
```

#### H37 – Proměnné, datové typy, vstup
```
Datové typy:
  int: 5, 100, -3
  float: 3.14, -0.5
  str: "Ahoj", "Franta"
  bool: True, False

jmeno = input("Jak se jmenuješ? ")
vek = int(input("Kolik ti je? "))
print("Ahoj", jmeno, "je ti", vek, "let")

Chyba: proč int(input(...))? → bez int je to text, nelze sčítat!
```

#### H38 – Podmínky (if / elif / else)
```python
vek = int(input("Věk: "))
if vek < 15:
    print("Jsi mladší")
elif vek == 15:
    print("Je ti přesně 15")
else:
    print("Jsi starší")

Projekt: Kalkulačka s výběrem operace (+, -, *, /)
         Ošetřit dělení nulou!
```

#### H39 – Smyčka for
```python
for i in range(5):        # i = 0, 1, 2, 3, 4
    print(i)

for i in range(1, 11):    # i = 1..10
    print(i, "×", 3, "=", i*3)

Projekt: Násobilka pro číslo zadané uživatelem
```

#### H40 – Smyčka while
```python
heslo = ""
while heslo != "tajne123":
    heslo = input("Zadej heslo: ")
print("Vítej!")

# Suma čísel dokud uživatel nezadá 0:
soucet = 0
cislo = 1
while cislo != 0:
    cislo = int(input("Číslo (0=konec): "))
    soucet += cislo
print("Součet:", soucet)
```

#### H41 – Funkce (def)
```python
def pozdrav(jmeno):
    print("Ahoj,", jmeno)

def secti(a, b):
    return a + b

vysledek = secti(5, 3)
print(vysledek)  # 8

Projekt: Funkce pro výpočet obsahu různých tvarů
  def obsah_obdelniku(a, b): return a * b
  def obsah_trojuhelniku(a, v): return a * v / 2
```

#### H42 – Seznamy (list)

```python
jmena = ["Anna", "Bořek", "Cyril"]

for jmeno in jmena:

Projekt: Třídní seznam – přidat, odebrat, výpis
```

#### H48–H49 – HTML: struktura webové stránky
```html

<!-- H49: seznamy, tabulky -->
```

#### H57 – Správa souborů, formáty, cloud + Modelování pomocí grafů
```
[Přípravy 15 + 16]

SPRÁVA SOUBORŮ, FORMÁTY A CLOUD (příprava 15):
  Typy souborů a jejich formáty:
    Dokumenty: .docx / .odt / .pdf / .txt / .md
    Obrázky: .jpg (ztrátová komprese) vs. .png (bezztrátová) vs. .svg (vektory)
    Videa: .mp4, .mkv; zvuk: .mp3, .wav, .flac
    Data: .csv, .json, .xml
    Archiv: .zip, .7z, .tar.gz

  Cloud úložiště:
    Google Drive, OneDrive, iCloud – výhody/nevýhody
    Synchronizace vs. záloha: co je co?
    Bezpečnost: dvoufaktorová autentizace, sdílení jen konkrétní osobě

  Praktické: Žáci organizují složku 10 smíšených souborů → přejmenovat, přesunout, zkomprimovat

MODELOVÁNÍ POMOCÍ GRAFŮ A SCHÉMAT (příprava 16):
  Graf (datová struktura): uzly + hrany
    Mapa metra = graf (stanice = uzly, linky = hrany)
    Přátelé na sociální síti = graf
  
  Praktické schémata:
    Vývojový diagram (opakování z 6.r. + komplex větve)
    Myšlenková mapa (MindMeister nebo papír)
    ER diagram (předvídání 8.r.)
  
  Aktivita: Nakresli graf "kdo s kým kamarádí ve třídě" → nalezni nejpropojené uzly
```
