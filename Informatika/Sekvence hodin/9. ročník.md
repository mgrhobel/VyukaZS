#### H33 – Vizualizace dat (matplotlib)
```python
import matplotlib.pyplot as plt

mesice = ["Led", "Úno", "Bře", "Dub", "Kvě", "Čer"]
teploty = [-2, 1, 6, 12, 17, 21]

plt.plot(mesice, teploty, marker="o", color="red")
plt.title("Průměrné teploty")
plt.xlabel("Měsíc")
plt.ylabel("Teplota (°C)")
plt.grid(True)
plt.show()

# Sloupcový graf
plt.bar(mesice, teploty, color="skyblue")
plt.show()
```

#### H34 – Web scraping základy (requests + BeautifulSoup)
