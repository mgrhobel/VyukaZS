#### H33 – Python: statistiky a datové analýzy bez externích knihoven
```python
import requests

# Stažení týdenní předpovědi počasí
url = "https://api.open-meteo.com/v1/forecast?latitude=50.07&longitude=14.42&daily=temperature_2m_max,temperature_2m_min&timezone=Europe/Prague&forecast_days=7"
r = requests.get(url)
data = r.json()

dny = data["daily"]["time"]
max_t = data["daily"]["temperature_2m_max"]
min_t = data["daily"]["temperature_2m_min"]

# Výpis jako textová tabulka (bez matplotlib)
print(f"{'Den':<12} {'Min':>6} {'Max':>6}")
print("-" * 26)
for den, t_min, t_max in zip(dny, min_t, max_t):
    print(f"{den:<12} {t_min:>5}°C {t_max:>5}°C")

# Statistiky v paměti (bez souborů, bez externích knihoven)
prumer = sum(max_t) / len(max_t)
print(f"\nPrůměrné max. teploty: {prumer:.1f} °C")
print(f"Nejtepleji: {max(max_t)} °C, Nejchladněji: {min(min_t)} °C")

# Bonus: seřadit dny dle teploty
serazene = sorted(zip(max_t, dny), reverse=True)
print("\nDny seřazené od nejteplejšího:")
for t, d in serazene:
    print(f"  {d}: {t} °C")

# Aktivita: Žáci stáhnou data svého města → porovnají s Prahou
# Výstup: textová tabulka + zkopírovat do Excelu → graf v Excelu
# 📎 Grafy přímo z Pythonu (matplotlib) → složka Střední Škola/ (SŠ 1.r. téma)
```

#### H34 – Zpracování dat: JSON a otevřená API
```python
import requests

# JSON = formát dat – stejná struktura jako Python slovník
# {
#   "daily": {
#     "time": ["2024-01-01", ...],
#     "temperature_2m_max": [5.2, ...]
#   }
# }

# Příklad 1: počasí – přístup k vnořeným klíčům
url = "https://api.open-meteo.com/v1/forecast?latitude=50.07&longitude=14.42&daily=temperature_2m_max,temperature_2m_min&timezone=Europe/Prague&forecast_days=7"
r = requests.get(url)
data = r.json()

dny = data["daily"]["time"]
max_t = data["daily"]["temperature_2m_max"]
min_t = data["daily"]["temperature_2m_min"]

for den, t_min, t_max in zip(dny, min_t, max_t):
    print(f"{den}: {t_min}°C – {t_max}°C")

# Příklad 2: data v paměti jako seznam slovníků (bez souborů)
zaci = [
    {"jmeno": "Anna", "body": 85, "trida": "9A"},
    {"jmeno": "Bořek", "body": 62, "trida": "9B"},
    {"jmeno": "Cecílie", "body": 91, "trida": "9A"},
]

# Filtrování a analýza – čistě v paměti
uspesni = [z for z in zaci if z["body"] > 70]
print(f"Úspěšných: {len(uspesni)}")

trida_9a = [z["body"] for z in zaci if z["trida"] == "9A"]
print(f"Průměr 9A: {sum(trida_9a)/len(trida_9a):.1f} b")

# Aktivita: Žáci si vyberou libovolné otevřené API (catfacts, open-meteo, …)
# a zpracují data v Pythonu – výstup jako textová tabulka nebo počítaná statistika

# Pojmy: JSON struktura, vnořené slovníky, zip(), list comprehension z API dat
# 📎 Ukládání dat do souborů (open/with + json.dump) → složka Střední Škola/ (SŠ téma)
```
