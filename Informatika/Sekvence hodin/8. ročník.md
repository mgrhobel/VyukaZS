#### H9 – Šifry a kryptografie (Pythonová kryptografie)
🧩 **PT: VDO** – VDO: šifrování jako nástroj ochrany soukromí, právo na bezpečnou komunikaci; historický kontext (válečná kryptografie)
```python
# Caesarova šifra
def zasifruj(zprava, posun):
    vysledek = ""
    for znak in zprava.upper():
        if znak.isalpha():
            # Posun v abecedě (modulo 26)
            zasifrovany = chr((ord(znak) - ord('A') + posun) % 26 + ord('A'))
            vysledek += zasifrovany
        else:
            vysledek += znak
    return vysledek

def desifruj(zprava, posun):
    return zasifruj(zprava, 26 - posun)

print(zasifruj("AHOJ SVETE", 3))   # DKRM VYHWH
print(desifruj("DKRM VYHWH", 3))  # AHOJ SVETE

# Lámání šifry – zkus všechny posuny
def lam_caesara(zasifrovano):
    for posun in range(26):
        pokus = desifruj(zasifrovano, posun)
        print(f"Posun {posun:2d}: {pokus}")

# Vigenèrova šifra (H9 rozšíření)
def vigenere(zprava, klic):
    vysledek = ""
    klic = klic.upper()
    i = 0
    for znak in zprava.upper():
        if znak.isalpha():
            posun = ord(klic[i % len(klic)]) - ord('A')
            zasifrovany = chr((ord(znak) - ord('A') + posun) % 26 + ord('A'))
            vysledek += zasifrovany
            i += 1
        else:
            vysledek += znak
    return vysledek
```

#### H10 – Turtle graphics + vizualizace (sloupcový graf)
```python
import turtle

t = turtle.Turtle()
t.speed(0)

for i in range(360):
    t.forward(i)
    t.right(59)

# Turtle sloupcový graf (bez rekurze)
def sloupcovy_graf(data, sirka=40, mezera=10):
    """data = [(popisek, hodnota), ...]"""
    screen = turtle.Screen()
    t = turtle.Turtle()
    t.speed(0)
    t.up()
    x = -len(data) * (sirka + mezera) // 2
    for popisek, hodnota in data:
        t.goto(x, 0)
        t.down()
        t.begin_fill()
        for _ in range(2):
            t.forward(sirka)
            t.left(90)
            t.forward(hodnota)
            t.left(90)
        t.end_fill()
        t.up()
        t.goto(x + sirka // 2, -20)
        t.write(popisek, align="center")
        x += sirka + mezera

mes = [("led", 30), ("úno", 50), ("bře", 80), ("dub", 110)]
sloupcovy_graf(mes)
turtle.done()
```

#### H31 – Micro:bit: první program – LED displej a animace *(příprava 20)*
```
Cíl: Žák spustí první program na Micro:bitu; ovládá LED displej a vytvoří animaci

  Přihlášení není nutné; uložit jako .hex soubor

LED displej 5×5:
  Blok "zobraz obrázek" → srdce, šipka, smajlík
  Blok "zobraz text" → scrollující text
  Blok "zobraz číslo"

Animace:
  "při spuštění" → opakuj stále → zobraz obrázek A → pauza → zobraz obrázek B

Vstup – tlačítka:
  "když je stisknuto A" → zobraz "Ahoj"
  "když je stisknuto B" → zobraz "Čau"
  "když je stisknuto A+B" → zobraz číslo 42

Projekt 1: Vytvoř animaci (min. 5 snímků) + reaguj na obě tlačítka
```

#### H32 – Micro:bit: proměnné, podmínky, senzory
```
Cíl: Žák používá proměnné a podmínky v Micro:bitu; čte hodnoty ze senzorů

Proměnné v Makecode:
  Vytvořit proměnnou "skóre"
  skóre = 0; při stisku A → skóre += 1; zobraz skóre

Podmínky:
  pokud skóre > 10 → zobraz hvězdu (hráč vyhrál)

Senzory Micro:bitu:
  Akcelerometr (náklon):
    "při zatřepání" → animace + zvuk
    Zjistit náklon ve směru X a Y (hodnota -1024 až 1024)
  Kompas (magnetometr):
    "při spuštění" → zobraz směr severu (hodnota stupňů)
  Teplotní senzor:
    "zobraz číslo teplota" → teplota v místnosti

Projekt 2: Digitální kostka
  Při zatřepání → náhodné číslo 1–6 → zobraz jako tečky nebo číslo
  Bonus: udrž skóre přes více hodů
```
