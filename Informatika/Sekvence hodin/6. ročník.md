## 6. ROČNÍK (~34 hodin)

### BLOK 1 – Digitální svět (5 hodin)

#### H1 – Úvod: Co je informatika a výpočetní myšlení
```
Cíl: Pochopit, že informatika ≠ práce s počítačem, ale způsob myšlení

Aktivity:
1. Anketa: Co si představíte pod slovem "informatika"? (lístečky)
2. Výpočetní myšlení – 4 pilíře:
   - Dekompozice (rozložit problém na části)
   - Abstrakce (zjednodušit, ignorovat nepodstatné)
   - Rozpoznávání vzorů (hledat opakující se struktury)
   - Algoritmizace (zapsat postup krok po kroku)
3. Příklad: Recept na palačinky = algoritmus!
4. CS Unplugged: Nepočítačová aktivita – seřadit čísla metodou bubble sort
```

#### H2 – Hardware a software: co je uvnitř počítače
```
Hardware: CPU, RAM, HDD/SSD, GPU, základní deska, periferie
Software: OS, aplikace, firmware

Aktivity:
- Puzzle: obrázky součástek → žáci pojmenují a přiřadí funkci
- Co se stane, když odstraním RAM? (diskuse)
- Rozdíl: rám kola (HW) vs. jízdní řád (SW) – analogie
```

#### H3 – Soubory, složky, souborový systém
```
Hierarchická struktura souborů
Přípony: .docx, .jpg, .mp3, .exe, .py
Praktické: organizace souboru → žáci vytvoří strukturu složek pro svůj projekt

Cvičení: Najdi v souborovém manažeru soubor "skrytý.txt" (nastavit zobrazení skrytých)
```

#### H4 – Internet: jak funguje, IP adresy, DNS
🧩 **PT: MV · VDO** – Mediální výchova: rozpoznávání důvěryhodných zdrojů (HTTPS vs. HTTP); VDO: právo na soukromí v síti
```
IP adresa = adresa v síti (jako domovní adresa)
DNS = telefonní seznam internetu (název → IP)
HTTP/HTTPS, URL, doménová jména

Nepočítačová demonstrace: Žáci = uzly sítě, posílají zprávy na lístečcích
Diskuse: Proč je HTTPS bezpečnější než HTTP?
```

#### H5 – Digitální stopa a bezpečnost
🧩 **PT: MV · VDO · OSV** – Mediální výchova: phishing, manipulace online; VDO: práva uživatele, GDPR úvod; OSV: osobní bezpečnost a odpovědné chování
```
Témata:
- Digitální stopa (co po nás zůstane na internetu)
- Silná hesla: délka > složitost; passphrase
- Phishing: jak poznat podvodný e-mail
- Soukromí a GDPR (základní pojem)

Aktivita: Test hesla na howsecureismypassword.net (ukázka – BEZ reálných hesel!)
Roleplay: Přišel ti e-mail "Vyhral jsi iPhone, klikni sem" – co uděláš?
```

#### 🧩 PT aktivita – Osobnostní a sociální výchova: „Digitální pohoda" (OSV)
> *Vhodné zařadit v návaznosti na H5 nebo jako samostatný blok na konci BLOK 1.*

**Cíl:** Žáci reflektují svůj vztah k digitálním technologiím, rozpoznají rizika nadměrného užívání a vytvoří si vlastní „Digitální kodex".

**Průběh (45 min):**
1. **Rozehřívací diskuse (10 min):** Kolik hodin denně strávíš na telefonu/počítači? Co nejčastěji děláš? Kdy ti to vadí (výuka, spánek, rodina)?
2. **Pojmy (10 min):** screen time, FOMO (fear of missing out), notifikační smyčka, deepfake, kyberšikana – žáci vysvětlí vlastními slovy
3. **Případové studie (10 min):** 3 situace v páru – žáci navrhnou řešení:
   - Jana dostává zprávy od neznámého, jsou stále agresivnější.
   - Tomáš zveřejnil fotku spolužáka bez dovolení – ten ho prosí, aby ji smazal.
   - Lenka hraje hry 5 hodin denně, nestíhá DÚ, ale nemůže přestat.
4. **Individuální práce (10 min):** Každý žák napíše svůj „Digitální kodex" (5 pravidel pro zdravé užívání internetu)
5. **Sdílení + reflexe (5 min):** Dobrovolníci přečtou pravidlo, které je pro ně nejdůležitější

**Výstup:** Digitální kodex (vyvěsit ve třídě nebo v EduPage)

### BLOK 2 – Algoritmy a programování v Scratch (20 hodin)

#### H6 – Scratch: prostředí a první projekt
```
Platforma: scratch.mit.edu (bez registrace nebo s školní)
Orientace:
  - Scéna, postavy (sprites), kostýmy, skripty
  - Blokové kategorie: Pohyb, Vzhled, Zvuk, Události, Řízení, Snímání, Operátory, Proměnné

Úkol: Rozhýbej postavu – klikni, chodí po scéně
Domácí: Přidej druhý kostým a animuj chůzi
```

#### H7 – Sekvence příkazů a debugging
🧩 **PT: OSV** – spolupráce v párech, rozvoj schopnosti sdílení práce a vzájemného hodnocení
```
Algoritmus = přesný postup = sekvence

Příklad: Nakresli čtverec (strana 100)
  otoč se o 90° → jdi 100 → otoč se 90° → jdi 100 → ... (4×)

Debugging – co když udělám špatný krok?
  - Záměrná chyba: otoč o 45° místo 90° → co se stane?
  - Práce v párech: jeden píše, druhý debuguje
```

#### H8 – Smyčka „opakuj N-krát"
```
Problém: Čtverec = 4× stejný kód → SMYČKA
  opakuj 4 krát: jdi 100; otoč se 90°

Rozšíření:
  - Pravidelný pětiúhelník (otoč o 72°), šestiúhelník (60°)
  - Spirála: každé opakování jdi o trochu dál

Projekt: Nakresli domeček (obdélník + trojúhelník střecha)
```

#### H9 – Větvení (podmínka „pokud")
```
Pseudokód:
  POKUD <podmínka>: dělej X
  JINAK: dělej Y

V Scratch:
  pokud <klávesa mezerník stisknuta?> → skoč
  pokud <dotýkáš se barvy červené?> → zastavit vše

Projekt: Hra – postava utíká, při dotyku překážky – stop
```

#### H10 – Smyčka „opakuj dokud" + nekonečná smyčka
```
Opakuj dokud <podmínka>:
  → hledání v textu, kontrola vstupu

Nekonečná smyčka + větvení = základ herní logiky
  opakuj vždy:
    pokud <klávesa doleva>: otoč se, jdi 10
    pokud <klávesa doprava>: otoč se, jdi 10

Projekt: Ovládaná postava + vyhýbání se překážkám
```

#### H11 – Proměnné
```
Proměnná = pojmenovaná krabička na hodnotu
Vytvoření v Scratch: Data → Vytvoř proměnnou → "skóre"

skóre = 0 (inicializace)
opakuj vždy:
  pokud <dotýkáš se hvězdy?>:
    zvyš skóre o 1
    hraj zvuk

Projekt: Collector game – sbírej hvězdy, vyhýbej se démonům, počítej skóre
```

#### H12 – Mini projekt č. 1: Jednoduchá hra
```
Zadání: Vytvoř hru, která má:
  ✓ Pohybující se postavu (šipky)
  ✓ Překážku nebo cíl
  ✓ Proměnnou skóre nebo životy
  ✓ Herní cyklus (začátek → konec)

Hodnocení (rubrika):
  2b – hra funguje
  2b – použita podmínka
  2b – použita proměnná
  2b – hra má smysl (lze ji hrát a porazit)
  2b – komentáře ve skriptech
  Max: 10 b
```

#### H13 – Vývojové diagramy
```
Symboly:
  ◯ Start / Konec
  ▭ Operace / Příkaz
  ◇ Podmínka (výběr, větvení)
  → Šipky (tok řízení)

Příklady:
  1. Vývojový diagram: "Vstanu ráno a jdu do školy"
  2. Jak ATM vydá peníze?
  3. Jak funguje if-else v Scratch → nakresli diagram

Aktivita: Žáci dostanou hotový diagram → naprogramují v Scratch
```

#### H14 – Parametry, klony (Scratch pokročile)
```
Klonování: jedna postava → mnoho kopií (nepřátelé ve hře)
  opakuj 5 krát: vytvoř klon

Zprávy (messages): komunikace mezi postavami
  pošli zprávu "startuj" → jiná postava reaguje

Projekt: Asteroid game – hráč střílí, klony asteroidů se pohybují
```

#### H15 – Práce s textem a scénáře
```
Proměnné pro text: ptej se → odpověď
Interaktivní příběh s větvením

Projekt: Dobrodružná hra (textová) – postava se ptá, hráč odpovídá, větví se příběh
Vazba na Čj: Žáci nejprve napíší scénář na papír, pak kódují
```

#### H16–H17 – Projekt č. 2: Scratch – volné téma + peer review
🧩 **PT: OSV** – prezentační dovednosti, vzájemné hodnocení (peer review), konstruktivní zpětná vazba
```
Žáci vytvoří projekt dle volby (hra, animace, interaktivní příběh, kvíz)

H16: Vývoj + průběžná kontrola (učitel obchází)
H17: Prezentace projektů: 2 min ukázka + co bylo nejtěžší
Peer review: každý hodnotí 2 spolužáky dle rubriky
```

#### H18 – Algoritmizace: třídění a vyhledávání
```
Nepočítačová demonstrace – CS Unplugged:
  Bubble sort: žáci jako čísla, přehazujeme sousedy
  Selection sort: najdi nejmenší, přesuň na začátek
  Binary search: hledáme ve slovníku – proč otevřít uprostřed?

Diskuse: Proč záleží na efektivnosti algoritmu?
```

#### H19–H20 – Scratch: Matematické projekty
```
H19: Kreslení geometrických tvarů pomocí Scratch
  - Pythagorova spirála (smyčka + rotace)
  - Koch snowflake pomocí smyčky (opakování úsečky + otočení – BEZ rekurze)
  - Animace: pohybující se sprite kreslí vzor
  📎 Pozn.: Rekurzivní fraktály (funkce volající sebe) jsou SŠ téma → složka Střední Škola/

H20: Scratch jako kalkulačka
  - Vstup: 2 čísla, operace
  - Výstup: výsledek
  - Rozšíření: ošetření dělení nulou
```

#### H21 – Písemka / Test č. 1 – Algoritmy a Scratch

3. (2b) Jaký je rozdíl mezi smyčkou "opakuj N-krát" a "opakuj dokud"?
4. (2b) Načrtni vývojový diagram pro: "Dokud nejsem doma, jdu dál."

ČÁST B – Scratch (6b):

### BLOK 3 – Tabulkový procesor a data (8 hodin)

#### H22 – Excel/Calc: buňky, vzorce, reference
```
Prostředí: LibreOffice Calc nebo MS Excel (dle školy)
  Buňka A1, B2... ; řádky, sloupce
  Vzorec: =A1+B1; =SUM(A1:A10); =AVERAGE(...)
  Absolutní reference: $A$1 (při kopírování se nemění)

Cvičení: Třídní statistika výsledků testu
  - Průměr třídy
  - Nejlepší výsledek (MAX)
  - Počet nad průměrem (COUNTIF)
```

#### H23 – Grafy v tabulkovém procesoru
🧩 **PT: MV** – Mediální výchova: kritické myšlení o grafech v médiích, manipulace s osou Y
```
Typy grafů a kdy je použít:
  Sloupcový: porovnání kategorií
  Spojnicový: vývoj v čase
  Koláčový: podíly z celku

Projekt: Žáci vytvoří graf z reálných dat (počasí, ceny potravin, výsledky her)
Diskuse: Jak lhát s grafy? (manipulace s osami)
```

#### H24 – Podmíněné formátování a funkce IF
```
=IF(podmínka; hodnota_true; hodnota_false)
=IF(A1>50; "Splnil"; "Nesplnil")

Podmíněné formátování: buňka červená pokud <50, zelená pokud >80
Projekt: Klasifikační arch – průměr + automatické known
```
