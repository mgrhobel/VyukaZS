## Měsíční rozvrh (počítáme s 2 h/týden)

### ZÁŘÍ–ŘÍJEN (cca 16 hodin)

**Celek 1 – Opakování** (6 h)
- Opakování algoritmů z 6.–7. ročníku (vývojové diagramy, Scratch)
- Přechod od blokového k textovému myšlení:
  - Jak instrukce v Scratchi odpovídají textu v Pythonu?
  - Přeložení jednoduchého Scratch skriptu do pseudokódu
- Instalace prostředí: Thonny IDE (zdarma, vhodné pro ZŠ) nebo VS Code / replit.com

**Celek 2 – Python: základní prvky** (10 h zahájeno v říjnu)

- Komentáře: `# toto je komentář`
- Proměnné: vytvoření, přiřazení hodnoty
- Datové typy: `int`, `float`, `str`, `bool`
- Vstup: `input("Zadej číslo: ")`; převod typů `int()`, `float()`, `str()`
- Aritmetické operátory: `+`, `-`, `*`, `/`, `//`, `%`, `**`

> 🧑‍🏫 **Tip:** Začni s Thonny – má integrovaný debugger „krok za krokem" ideální pro pochopení toku programu.

### LISTOPAD–PROSINEC (cca 14 hodin)

**Celek 2 – Python: základní prvky** (dokončení, 6 h)
- Podmínky: `if`, `elif`, `else`
- Porovnávací operátory: `==`, `!=`, `<`, `>`, `<=`, `>=`
- Logické operátory: `and`, `or`, `not`
- Vnoření podmínek

**Celek 3 – Python: cykly** (8 h)
- Cyklus `for` s `range()`: `for i in range(10):`
- Cyklus `while`: podmíněné opakování
- `break` a `continue` – přerušení a přeskočení iterace
- Vnořené cykly (kreslení tabulek, vzorů)
- **Mini-projekt:** Hádání čísla (počítač náhodně vybere 1–100, uživatel hádá)

```python
# Ukázka: hádání čísla
import random
tajne = random.randint(1, 100)
hadani = 0
while hadani != tajne:
    hadani = int(input("Hádej číslo (1-100): "))
    if hadani < tajne:
        print("Víc!")
    elif hadani > tajne:
        print("Míň!")
print("Správně! 🎉")
```

### LEDEN–ÚNOR (cca 16 hodin)

**Celek 3 – Python: funkce a řetězce** (12 h)
- Funkce: definice `def`, parametry, návratová hodnota `return`
- Scope proměnných (lokální vs. globální – přehled)
- Vestavěné funkce: `len()`, `abs()`, `round()`, `max()`, `min()`
- Řetězce (string):
  - Indexování a řezy: `s[0]`, `s[1:4]`, `s[-1]`
  - Metody: `.upper()`, `.lower()`, `.strip()`, `.replace()`, `.split()`
  - f-řetězce: `f"Jmenuji se {jmeno}"`
- Seznam (list):
  - Vytvoření, indexování, řez
  - Metody: `.append()`, `.remove()`, `.sort()`, `.len()`
  - Iterace přes seznam: `for prvek in seznam:`

**Celek 4 – Algoritmy a složitost** (4 h zahájeno v únoru)
- Třídící algoritmy: Bubble sort (implementace v Pythonu)
- Vyhledávací algoritmy: lineární hledání vs. binární hledání (přehled)
- Pojem složitosti: proč záleží na počtu kroků?

> 📝 **Pololetní projekt:** jednoduchý program v Pythonu (kalkulačka, kvíz, filtrování seznamu).

### KVĚTEN–ČERVEN (cca 14 hodin)

**Celek 7 – Základy webu: HTML a CSS** (10 h)
- Jak funguje web: klient–server, HTTP, URL, DNS (přehled)
- HTML – struktura stránky:
  ```html
  <!DOCTYPE html>
  <html lang="cs">
    <head><meta charset="UTF-8"><title>Moje stránka</title></head>
    <body>
      <h1>Nadpis</h1>
      <p>Odstavec textu.</p>
      <a href="https://example.com">Odkaz</a>
      <img src="obrazek.jpg" alt="Popis obrázku">
    </body>
  </html>
  ```
- Základní tagy: `<h1>`–`<h6>`, `<p>`, `<a>`, `<img>`, `<ul>`, `<ol>`, `<li>`, `<table>`
- CSS – propojení se stránkou; základní vlastnosti:
  ```css
  body { background-color: #f0f0f0; font-family: Arial, sans-serif; }
  h1 { color: #333; text-align: center; }
  ```
- **Projekt:** Jednoduchá osobní webová stránka (1–3 stránky s CSS)

**Celek 8 – Digitální bezpečnost** (2 h – zopakování a prohloubení)
- Šifrování: symetrické vs. asymetrické (přehled, HTTPS)
- GDPR: co to je, jaká máme práva
- Správce hesel: proč a jak používat

**Celek 9 – Informační společnost** (2 h)
- Dezinformace a fake news: jak rozpoznat, jak ověřit
- Mediální gramotnost: kdo píše, proč, komu to slouží
- Algoritmické bubliny (filter bubble) na sociálních sítích
- Dopady AI na společnost (přehled)

**Celek 10 – Závěrečný projekt** (10 h – průběžně)
- Žáci si zvolí zaměření:
  - **A: Python projekt** – větší program (hra, kalkulačka s GUI, datová analýza)
  - **B: Web projekt** – vícestránková stránka s HTML/CSS (event. s JS efekty)
  - **C: Datový projekt** – analýza reálného datasetu v Excelu/Sheets + prezentace výsledků
- Prezentace projektu (5 minut) + technická dokumentace (README)
