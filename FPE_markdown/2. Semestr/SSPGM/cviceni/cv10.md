2.
Napiste program, kde forkem spustite proces potomek.

master resp. slave dvakrat po sobe zapisovat znak (zadany znak zadavejte
jako parametr prikazove radky), resp. cist 10 znaku v nasledujici
casove posloupnosti:

rodic:
t1 write1 t2 write2

potomek:
t3 read1 t4 read2

Casova zpozdeni zadavejte jako argumenty prikazove radky a realizujte
volanim sleep(t1) ....
Do tabulky viz nize zobrazte sekvenci jednotlivych read1, read2, write1,
write2 pri zmene casovych zpozdeni t1, t2 , t3, t4.

WRITE   |   READ
t1|t2   |   t3|t4
-----------------------------------------------------------------------
0  0    |   0  0	write1 read1 write2 read2
0  2    |   0  0	write1 read1 read2  write2
.....

errno.h, stdio.h, fcntl.h

Ke konvertovani stringu na int pouzijte funkci int atoi(const char *nptr).

Analyzujte dle zadavanych kombinaci t1, t2, t3, t4
a obsahu zmodifikovaneho souboru p14.txt.
