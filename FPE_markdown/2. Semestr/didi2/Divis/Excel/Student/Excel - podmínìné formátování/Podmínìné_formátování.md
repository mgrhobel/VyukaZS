# *Podmíněné formátování* {#podmíněné-formátování .western lang="cs-CZ" style="margin-top: 0.08in"}

***Formát*** -- ovlivňuje jak budou data v buňce zobrazena

*Číselný* - ovlivňuje způsob zobrazení dat. Nastavuje se v menu Formát 
Buňky...\
 karta Číslo

Každé číslo použité v Excelu má dvě části:

-   -   **Hodnotu -** ta se zapisuje

    -   **Formát --** ten se volí předem nebo dodatečně

**Příklad:** Hodnota čísla: 28,25

Výsledné formáty:

*28*

*28,250*

*28,25 Kč*

*2,83E+01*

*6:00 hod*

*28.leden 1900*

*Stylistický* - přispívá k lepšímu vzhledu listů (stínování, okraje,
barvy apod.). Nastavuje se v menu Formát  Buňky...  karty Zarovnání,
Písmo, Ohraničení a Vzorky

*Podmíněné formátování* -- umožňuje buňku automaticky zformátovat, pokud
byla splněna předem nadefinovaná podmínka (více podmínek), která může
být dána vzorci

## Postup při nastavování automatického formátování: {#postup-při-nastavování-automatického-formátování .western lang="cs-CZ"}

1.  Výběr buněk či oblasti

2.  Formát  Podmíněné formátování

3.  Výběr požadovaného typu podmíněného formátu

```{=html}
<!-- -->
```
1.  -   *Hodnota buňky* - zformátujete buňku na základě její hodnoty
        nebo obsahu

    -   *Vzorec* - zformátujete buňku na základě zadaného vzorce. Zadaný
        vzorec musí mít pravdivou nebo nepravdivou hodnotu

```{=html}
<!-- -->
```
4.  Výběr operátoru - nastavíme podmínku. Klepneme-li na možnost „Je
    mezi" nebo „Není mezi", musíme zadat horní a dolní hodnotu
    pro podmínku. Horní a dolní hodnota je zahrnuta do oblasti.

5.  Nastavení výsledného formátu -- jak bude buňka vypadat při splnění
    podmínky. Můžeme nastavit:

```{=html}
<!-- -->
```
1.  -   Písmo - řez písmo (tučné, kurzíva) barvu písma, podtržení a
        přeškrtnutí

    -   Ohraničení -- vzhled ohraničení formátované oblasti

    -   Vzorky -- barva stínování a vzorek

```{=html}
<!-- -->
```
6.  Přidat další podmínku -- lze zadat až tři podmínky u jednoho
    podmíněného formátování
