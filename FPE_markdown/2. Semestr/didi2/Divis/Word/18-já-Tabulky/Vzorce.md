+----------------+----------------------------------------------------+
| ABS(*x*)       | Kladná hodnota čísla nebo vzorce bez ohledu na     |
|                | jeho skutečnou kladnou nebo zápornou hodnotu.      |
+----------------+----------------------------------------------------+
| AND( *x*,*y*)  | Hodnota 1, jsou-li oba logické výrazy *x* a *y*    |
|                | pravdivé, nebo hodnota 0 (nula), pokud je jeden z  |
|                | výrazů nepravdivý.                                 |
+----------------+----------------------------------------------------+
| AVERAGE( )     | Průměr seznamu hodnot.                             |
+----------------+----------------------------------------------------+
| COUNT( )       | Počet položek v seznamu.                           |
+----------------+----------------------------------------------------+
| DEFINED(*x*)   | Hodnota 1 (pravda), pokud je výraz *x* platný,     |
|                | nebo hodnota 0 (nepravda), jestliže výraz nelze    |
|                | počítat.                                           |
+----------------+----------------------------------------------------+
| FALSE          | 0 (nula).                                          |
+----------------+----------------------------------------------------+
| INT(*x*)       | Čísla hodnoty nebo vzorce *x*, která leží nalevo   |
|                | od desetinné čárky.                                |
+----------------+----------------------------------------------------+
| MIN( )         | Nejmenší hodnota v seznamu.                        |
+----------------+----------------------------------------------------+
| MAX( )         | Největší hodnota v seznamu.                        |
+----------------+----------------------------------------------------+
| MOD(*x*,*y*)   | Zbytek dělení hodnoty *x* celým číslem *y*.        |
+----------------+----------------------------------------------------+
| NOT(x)         | Hodnota 0 (nula) (nepravda), je-li logický výraz   |
|                | *x* pravdivý, nebo hodnota 1 (pravda), pokud je    |
|                | výraz nepravdivý.                                  |
+----------------+----------------------------------------------------+
| OR(x,y)        | Hodnota 1 (pravda), pokud jsou jeden nebo oba      |
|                | logické výrazy *x* a *y* pravdivé, nebo hodnota 0  |
|                | (nula) (nepravda), jestliže jsou oba výrazy        |
|                | nepravdivé.                                        |
+----------------+----------------------------------------------------+
| PRODUCT( )     | Výsledek násobení seznamu hodnot. Například        |
|                | funkce\                                            |
|                | **{** = PRODUCT (1,3,7,9) **}** vrátí výsledek     |
|                | 189.                                               |
+----------------+----------------------------------------------------+
| ROUND(*x*,*y*) | Hodnota argumentu *x* zaokrouhlená na zadaný počet |
|                | desetinných míst *y*. Argument *x* může být číslo  |
|                | nebo výsledek vzorce.                              |
+----------------+----------------------------------------------------+
| SIGN(*x*)      | Hodnota 1, pokud má *x* kladnou hodnotu, nebo      |
|                | hodnota --1, má-li x zápornou hodnotu.             |
+----------------+----------------------------------------------------+
| SUM( )         | Součet seznamu hodnot nebo vzorců.                 |
+----------------+----------------------------------------------------+
| TRUE           | Hodnota 1.                                         |
+----------------+----------------------------------------------------+
