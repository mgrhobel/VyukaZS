# Zadání:

**Úkol č. 2:**

\

1.     Určitě jste se setkali s projevy koncepcí vyučování: výuka
orientovaná na učivo a učitele, výuka orientovaná na žáka? Rozmyslete
si, zda jste se setkali s oběma či nikoliv,své názory zdůvodněte.

2.     Své zkušenosti popište**.**

3.     Vypracování odešlete tutorovi.

\

\

Poznámka k bodu 2:

*Bylo by dobré nejdříve popsat svou zkušenost a pak koncepci vyučování,
se kterou jste se setkali. V popisu koncepce je dobré uvést: cíle,
postavení žáka a učitele, obsah učiva, popř. pojmenovat svým způsobem
používané metody, formy, podmínky výuky, apod.*
