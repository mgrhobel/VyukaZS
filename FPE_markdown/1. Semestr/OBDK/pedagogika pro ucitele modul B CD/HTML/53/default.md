# Zadání:

**Úkol č. 3**

***Zformulujte konkrétní vzdělávací cíl vybraného tématu Vašeho
odborného zaměření. Při formulaci užijte také Bloomovu taxonomii
cílů.***\

\

**1.     Vyberte si téma z vašeho odborného zaměření (např. Stavba
kosterního svalu, Předložky se 4. pádem., Personal pronouns, Výpočet
objemu těles, atd. prostě dle vašeho zaměření a přání).**

**2.     Stanovte si obsah učiva zvoleného tématu a aplikujte postup
formulace cílů na toto vybrané téma (pokud si nejste jisti, využijte
přílohu č.1 a text a příklady ve studijním článku 2.: Formulace cílů ve
vyučování.)**

**3.     Písemně zaznamenejte Vámi formulovaní cíl daného tématu.**

**4.     Odešlete tutorovi.**\
