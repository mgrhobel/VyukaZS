# Zadání:

***Aktivita č. 1***

\

**1.     Opatřete si pedagogickou dokumentaci jako: Rámcový vzdělávací
program pro ZV. Dále pak učební osnovy a učební plán.**

**2.     Vyhledejte v uvedených dokumentech obecné cíle   - vzdělávací,
výchovné a psychomotorické (dovednostní).**

**3.     Pak si vyberte Vámi oblíbené učební téma a pokuste se v něm
vysledovat konkrétní cíle. Pokud je v dokumentech nenajdete - zkuste se
zamyslet a cíle si vytvořte sami.**

**4.     Odešlete tutorovi.**

\
