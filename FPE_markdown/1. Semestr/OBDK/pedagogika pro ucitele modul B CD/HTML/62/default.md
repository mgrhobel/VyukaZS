# Zadání:

**Úkol č. 2. Srovnávací studie k hodnocení žáků na ZŠ a SŠ - analýza
dokumentů.**

\

1.     Vyhledejte na webových stránkách základních a středních škol v ČR
(nebo přímo navštivte různé školy ve svém regionu) různé podoby
klasifikačních řádů a různé formy hodnocení žáků.

2.     Analyzujte je a porovnejte kritéria stanovená pro hodnocení žáků.

3.     Závěry z analýzy sepište, uveďte shody a rozdíly, zaznamenejte
také vlastní postřehy a nápady. Písemnou zprávu můžete opatřit také
zajímavými přílohami s ukázkami hodnocení žáků.

4.     Odešlete tutorovi.

\
