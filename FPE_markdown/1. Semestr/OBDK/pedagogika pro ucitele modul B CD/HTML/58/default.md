# Zadání:

**Úkol č. 1.**

**Aplikace výukových strategií v praxi školy**

\

1.     Vyzkoušejte si na základní nebo střední škole aplikaci vybrané
výukové metody a organizační formy vyučování.

2.     Po praktické realizaci metody a formy zhodnoťte její přínos pro
danou vyučovací jednotku.

3.     Navrhněte případné změny z pohledu učitele a žáků.

4.     Zpracujte závěrečnou písemnou zprávu z vaší praxe..

5.     Odešlete tutorovi.
