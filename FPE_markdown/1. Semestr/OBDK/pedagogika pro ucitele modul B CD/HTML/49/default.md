# Zadání:

**Úkol č. 3:**

\

**Průzkum v praxi školy**

1.     Vyberte si školu (základní, střední, gymnázium, atd.- podle svého
oborového zaměření) ve vašem regionu. Domluvte se na této vybrané škole
s některým z učitelů (nejlépe s třídním učitelem).

2.     Při návštěvě školy a učitele zjistěte, jakou pedagogickou
dokumentaci učitelé využívají a jaký je její účel.

3.     Dokumentaci pak rozdělte na teoretickou a praktickou dokumentaci.

4.     Porovnejte svá zjištění (zejména u praktické dokumentace) na
škole se standardy povinné pedagogické dokumentace, kterou předepisuje
ministerstvo školství (např. využijte webové stránky ministerstva).

5.     Odešlete tutorovi.

\
