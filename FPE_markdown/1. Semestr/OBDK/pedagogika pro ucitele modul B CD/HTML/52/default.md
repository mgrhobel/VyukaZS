# Zadání:

**Aktivita č. 2**

***Pokuste se o konkretizaci a aktualizaci cílů tematických celků
jednotlivých ročníků vzhledem k podmínkám vaší školy a žákům v dané
skupině.***

\

**1.     Sežeňte si učební plán pro 2. stupeň ZŠ (6. - 9. ročník).**

**2.     Vypište si všechny tématické učební celky podle ročníků .**

**3.     K jednotlivým tematickým celkům v jednotlivých ročnících
navrhněte obecné cíle.**

**4.     Zamyslete se, zda učivo tématických celků a cíle jednotlivých
ročníků na sebe navazují a vedou k obecnému cíli - profilu absolventa
Vaší školy.**

**5.     Odešlete tutorovi.**
