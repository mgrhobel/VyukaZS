# Zadání:

**Úkol č. 1. Analýza hodnocení v platných školských dokumentech**

\

1.     Vyhledejte na webových stránkách (MŠMT ČR a VÚP Praha)
dokumenty - platný školský zákon, RVP pro základní (event. střední )
vzdělávání.

2.     Vyhledejte a analyzujte v uvedených dokumentech pohledy na proces
hodnocení žáků.

3.     Závěry z analýzy sepište a doplňte svými postřehy a nápady.

4.     Odešlete tutorovi.
