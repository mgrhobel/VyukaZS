# Zadání:

**Úkol č. 4 -** ![tužka](tuzka.jpg)\

**Volitelné úkoly na závěr kapitoly o cílech. Vyberte si 1 úkol z
následujících úkolů:**

\

**1.     Projekt vyučovací hodiny s konkretizací vzdělávacího
(kognitivního cíle) zvoleného tématu.**

**2.     Projekt odborné exkurze s konkretizací cílů pro studenty a
učitele.**

**3.     Projekt školní slavnosti s konkretizací cílů pro učitele,
studenty, rodiče, popř. veřejnost.**\
\

**Doporučení:**

· **      Výběr jednoho úkolu podle svého uvážení**

· **      Zpracování - vlastní volba a návrhy, písemné vyhotovení**

· **      Odeslání mailem či poštou**

· **      Vítána je tvořivost a originalita zpracování.**\
