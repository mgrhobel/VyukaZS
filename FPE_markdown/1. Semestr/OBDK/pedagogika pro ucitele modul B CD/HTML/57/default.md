# Zadání:

**Úkol č. 3. Návrhy na motivaci žáků (kooperace ve skupinách minimálně 3
členných)**

\

1.     Vytvořte si tříčlenné skupinky. Společně se domluvte na výběru
rozvíjení a aktualizace žákových motivačních potřeb (např. poznávacích,
výkonových, sociálních).

2.     Navrhněte způsoby rozvoje vybraného druhu motivace žáků ve
vyučování vašich aprobačních předmětů. Buďte konkrétní.

3.     Své návrhy zkonzultujte s učitele z praxe nebo didaktikem
odborného předmětu.

4.     Závěry zpracujte a odešlete tutorovi.
