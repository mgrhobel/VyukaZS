# Zadání:

**Úkol č. 2. Každý si zaslouží podporu (kooperace ve dvojicích)**

\

1.     Domluvte se s jedním z vašich kolegů a zkuste ve dvojicích
vymyslet slova, věty, které by žáky povzbudily k další činnosti.

2.     Vaše návrhy sepište.

3.     Následně si vyměňte zkušenosti s ostatními studenty (mailem,
chatem, v diskusním fóru, apod.)

4.     Výsledky zpracování a výměny zkušeností odešlete tutorovi.
