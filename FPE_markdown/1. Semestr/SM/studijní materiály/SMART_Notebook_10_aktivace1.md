Aktivace software SMART Notebook 10
Pro plnou funkčnost software SMART Notebook 10 je nutné tento software aktivovat produktovým klíčem.
Produktový klíč je nutné získat vyplněním formuláře na webových stránkách SMART Technologies (přesný
postup je popsán níže v tomto návodu). Získaným produktovým klíčem je následně možné aktivovat více
instalací SMART Notebook 10, není tedy nutné generovat nový klíč pro každou instalaci software. Aktivace
neprobíhá online, počítač na kterém je software právě aktivován nemusí být připojen k Internetu. Aktivace
je možná buď zadáním tohoto klíče přímo při instalaci software, nebo kdykoliv později pomocí Uvítacího
centra (Welcome Center).
Neaktivujete-li SMART Notebook 10 do 30-ti dní od instalace, software se zablokuje. Zablokování je
možné zrušit aktivací produktu.

Aktivace software

1) Aktivace software při instalaci
Zadáte-li produktový klíč při instalaci software SMART Notebook 10, bude nainstalovaný software
automaticky bezobslužně aktivován při prvním spuštění software.

2) Aktivace software z dialogu Zkušební verze:
Při každém spuštění neaktivovaného software se vždy zobrazí dialog Zkušební verze s výzvou k
aktivaci. Kliknutím na tlačítko Aktivovat produkt spustíte průvodce aktivací:
