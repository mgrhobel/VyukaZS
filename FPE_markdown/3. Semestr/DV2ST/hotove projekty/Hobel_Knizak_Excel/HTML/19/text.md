# Vzorce

## Numerické výpočty ze základní matematiky

Otevřete si nový sešit a proveďte následující výpočty:

123 - 87

(18 + 6) · 13

18 · 13 + 6 · 13

11^2^

(16 + 13) : 3

41 + (14 · 3) - (31 : 0,6)

5,3 · 3,5 + 5^3^ - 3^5^

(4,2 + 16 : 3) : (14,8 - 11 · 0,4)

Sešit uložte pod názvem \"**Vypocty**\".

Pomůcka: Pozor na podobu matematických úkonů pro počítač:

\

::: {align="center"}
  --------- ----
     \+      \+
     \-      \-
      ·      \*
      :      /
   mocnina   \^
     ()      ()
  --------- ----
:::

\

Znaménko umocnění \^ vložíte klávesovou zkratkou **Pravý Alt+3**
(zobrazí se však až po stisku další klávesy!) (Na anglické klávesnici
**Shift+6**)

\

***Úkol pro rychlejší žáky:***

*Do vedlejšího sloupce vypočtěte dvojnásobky hodnot ze sloupce A.*
