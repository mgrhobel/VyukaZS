# Podmíněné formátování

## Tělesná teplota pacientů

Otevřete soubor \"**Pacienti**\".

V tabulce je soupis naměřené tělesné teploty při ranní vizitě.
Potřebujeme, aby bylo na první pohled vidět, kdo má teplotu normální (do
37°C), kdo zvýšenou (37°C až 39°C) a kdo horečku (nad 39°C). Použijte
podmíněné formátování.

Uložte změny v souboru \"**Pacienti**\".

\

*Pomůcka:*

*Excel 2003: Formát -\> Podmíněné formátování\...*

*Excel 2007: Karta Domů -\> Podmíněné formátování -\> Nové pravidlo* -\>
*Formátovat pouze buňky obsahující*

*Správně vyplnit pravidla a nastavit formát.*
