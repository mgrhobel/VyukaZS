# Výuka Excelu na ZŠ

Tento výukový materiál je určen pro učitele informatiky na základních
školách. Obsahuje příklady pro výuku Excelu, které může učitel
informatiky použít při hodinách. Najde zde vždy zadání, pracovní soubor
a soubor se vzorovým řešením.

## Cíle

Pomocí tabulkových kalkulátorů můžeme aplikovat matematické znalosti do
informatiky. Používáme matematické výrazy, funkce agrafy. Použití funkcí
posiluje matematickou představu.

-   Ovládat tabulkový procesor, orientovat se v jeho prostředí, umět
    využít nejdůležitější funkce, porozumět novým pojmům - adresa,
    vzorec, funkce, formát; řešit úlohy na zpracování dat.
-   Zpracovávat data a údaje, rychle zadávat údaje pomocí automatického
    vyplňování, provádět jednoduché výpočty pomocí vzorců a funkcí,
    upravit tabulky do publikovatelné podoby a data vynést do grafu,
    orientovat se v rozsáhlejších tabulkách.
-   Pochopit princip činnosti tabulkového kalkulátoru - po změně
    vstupních dat se přepočítají všechny výsledky\
    \

**Formy výuky:**

-   Řešení matematický příkladů s pomocí PC.
-   Zpracování dat - práce s rozsáhlejší tabulkou dat.
-   Rozvíjení logického myšlení žáků.
-   Rychlé zpracovná velkého množství výpočtů.\
    \

**Použitá literatura:**

-   Blažíčková, Š: Microsoft Excel pro školy - cvičebnice, CP Books,
    a.s. Brno, 2005, ISBN: 80-251-0502-4
-   Vaníček, J: Metodická příručka - Informatika pro základní školy a
    víceletá gymnázia, CP Books, a.s. Brno, 2005, ISBN: 80-251-0631-4
