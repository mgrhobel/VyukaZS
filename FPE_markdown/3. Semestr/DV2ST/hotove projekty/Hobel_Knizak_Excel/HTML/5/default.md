# Úvod do Excelu

## Předpokládané vstupní znalosti

Pro práci s touto metodickou příručkou se předpokládají zejména tyto
vstupní znalosti a dovednosti žáků získané v předchozí výuce
informatiky:

-   Práce s klávesnicí a myší (umět napsat libovolný znak i ten, který
    není přímo na klávesnici)
-   Orientace v MS Windows - zejména v adresářové struktuře
-   Spustit potřebný program
-   Otevírání a ukládání souborů (Soubor -\> Otevřít; Soubor -\> Uložit
    / Uložit jako\...)
-   Práce se schránkou (Ctrl+C, Ctrl+V)
