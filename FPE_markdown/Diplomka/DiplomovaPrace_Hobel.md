Podpora internacionalizace a lokalizace aplikace MPA

Cílem práce je navrhnout a implementovat grafický nástroj, který bude usnadňovat procesy internacionalizace a lokalizace aplikace MPA (Medical Process Assistant).

Seznamte se s obecnými lokalizačními metodikami, které Java podporuje:

- Oddělení jazykově závislých dat od zdrojových kódů.

- Formátování číselných a časových datových typů v různých jazycích.

- Problematika speciálních znaků v různých jazycích.

Vytvořený nástroj bude umožňovat export všech textů z MPA do Hypersonic databáze, jejich editaci, sjednocení a následný import zpět do MPA.

- Pro implementaci grafického rozhraní použijte Swing.

- Srovnejte s postupy před vznikem tohoto nástroje a zhodnoťte jeho přínos.
