Popis postup :
(Ozna ení bod v obrázcích by bylo velmi zdlouhavé, proto chybí. Vrcholy krychle jsou pojmenovány
obvyklým zp sobem.)
1. a. P ímka ES leží v rovin EAS. Pr se nicí rovin DBF a EAS je p íka q (viz obrázek). Hledaný pr se ík
je pr se íkem p ímek ES a q.
b. P ímka DF leží v rovin DBF. Pr se nicí rovin EBG a DBF je p ímka q (viz obrázek). Hledaný
pr se ík je pr se íkem p ímek DF a q.
c. ezem krychle a roviny je lichob žník ASUT. P ímka RV leží v rovin RBF. Pr se nicí rovin ATU a
RBF je p íka q (viz obrázek). Hledaný pr se ík je pr se íkem p ímek RV a q.
2. a. Spojíme body E a H. Spojíme body E a M. Pr se íkem p ímky EM a hrany BF je bod Q. Bodem H
vedeme rovnob žku s p ímkou EM. Pr se íkem této p ímky s hranou CG je bod R. ezem je obdélník
EHRQ.
b. Spojíme body N a O. Pr se ík p ímky NO s hranou HD je bod U. Spojíme body O a T. Pr se íkem
p ímky OT a hrany HG je bod V. Pr se íkem p ímky OT a hrany FG je bod W. Spojíme body N a T.
Pr se íkem p ímky NT a hrany BF je bod X. Hledaným ezem je p tiúhelník NUVWX.
3. a. Spojíme body E a P. Bodem Q vedeme rovnob žku s p ímkou EP. Její pr se ík s hranoi BC
ozna íme R. Hledaným ezem je lichob žník EPRQ.
b. P ímka QG je rovnob žná s rovinou ezu a také s rovinou st ny ADHE, je tedy rovnob žná s jejich
pr se nicí. Pr se nice prochází bodem H a je rovnob žná s p ímkou QG. Pr se ík rovnob žky
s p ímkou QG vedené bodem H s hranou AE je bod R Bodem R vedem rovnob žku s p ímkou HC. Její
pr se ík s hranou AB ozna íme S. Spojíme body RC. Výsledkem je lichob žník HCRS.
4. a. Pr se ík p ímky PQ a p ímky BC ozna íme S. Spojíme body G a S. Pr se ík p ímky GS a hrany BF
ozna íme T. Pr se ík p ímky PQ a CD ozna íme U. Spojíme body G a U. Pr se ík P ímky GU a hrany
DH ozna íme V. Výsledkem je p tiúhelník QTGVP.
b. Pr se ík p ímek RS a BF ozna íme T. Spojíme body C a T. Pr se ík p ímky CT a hrany FG
ozna íme U. Pr se ík p ímek RS a BA ozna íme V. Spojíme body C a V. Pr se ík p ímek CV a AD
ozna íme W. Výsledkem je p tiúhelník CURSW.
