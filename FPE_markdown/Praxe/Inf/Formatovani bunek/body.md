*Formatovani bunek
- zlepseni zobrazeni dat a vypoctu.
- definujeme jim zobrazeni dat v bunce.
- muzeme urcit nejen zobrazeni udaju, ale take ohraniceni nebo pozadi bunky.

Priklady
-> format mena (cislo v Kc), pokud se vynasobi -> pak da spravny vysledek (bez Kc).
-> zminit prikald se zaokrouhlovanim - funkce a format cislo vypadaji stejne, ale po vynasobeni davaji jiny vysledek.

*Proc teda formatovat?
- zprijemneni pohledu na data a priprava pro jejich prezentaci.
- klicove informace se mohou zduraznit a stat se tak vice zrejmejsi.


*Z formatem jste se jiz pravdepodobne setkali
- zvetseni velikosti sloupce, aby bylo cislo videt.


*Ruzne typy formatovani - moznosti jak formatovat a co lz formatovat.
- pouzit priklad Formaty.xls

Jak:
- pridanim specialniho symbolu
- pomoci panelu format
- Vice moznosti pro format dialogove okno: Format -> Bunky
Vsimnout si, ze se zde nachazi i velikost radku a sloupce, jak jsem rikal, ze se s tim uz setkali.
(- prave tlacitko -> format bunek.)



*Ciselny format bunky 
- jak cislo bude zobrazeno je pouze vlastnost !bunky! - nikoliv cisla. Pokud se v bunce smaze nebo nahradi jinym, bude zobrazzeno ve stejnem formatu jako cislo predchozi.



*Poznamky
- nejprve je vhodne tabulku naplnit daty (to uz mame) a az pote zformatovat -> vyhneme se tak problemum z kopirovanim formatu (pri kopirovani se automaticky kopiruje i format tech dat, pokud nepouzijeme VLozit jinak, kde se da zakazat kopirovani formatu).

Procenta. 
- Namisto pouziti vzorce je mozne pracovat s hodnotou bunky, ktera je ve formatu procent. To je nejlepsi zpusob, rozviji hlavni myslenku excelu - z kazdou zmenu se vysledek prepocita. -> pakticky ukazat proc je 2. priklad lepsi nez prvni.
